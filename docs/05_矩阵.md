# LeetCode Hot 100 - 05 矩阵 C++ 题解

## 一、矩阵知识点回顾

矩阵题本质是二维数组遍历。关键在于别迷路：要么用标记记录行列状态，要么用边界控制遍历范围，要么利用行列单调性排除搜索空间。

常用写法：

```cpp
int m = matrix.size();
int n = matrix[0].size();
```

常见技巧：

- 第一行第一列做标记。
- 上下左右四个边界收缩。
- 转置 + 翻转完成旋转。
- 从右上角或左下角搜索有序矩阵。


## 二、基础容器/语法解释

矩阵在 C++ 中通常写作：

```cpp
vector<vector<int>> matrix;
```

### 获取行列数

```cpp
int m = matrix.size();
int n = matrix[0].size();
```

- `m`：行数。
- `n`：列数。

### 访问元素

```cpp
matrix[i][j]
```

表示第 `i` 行、第 `j` 列元素，下标从 0 开始。

### 遍历矩阵

```cpp
for (int i = 0; i < m; i++) {
    for (int j = 0; j < n; j++) {
        // matrix[i][j]
    }
}
```

### reverse 一行

```cpp
reverse(matrix[i].begin(), matrix[i].end());
```

用于将某一行原地反转。

## 1. 矩阵置零

链接：https://leetcode.cn/problems/set-matrix-zeroes/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

根据矩阵中的 0，把对应整行和整列置零。

### 这题考什么

考察原地标记和避免标记污染。

### 生活化理解

像表格里某个格子异常，整行整列都要标红，但不能边检查边乱涂，否则会影响后续判断。

### 核心思路

用第一行和第一列作为标记数组，额外记录第一行、第一列本身是否需要置零。

### C++ 代码

```cpp
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int m = matrix.size();
        int n = matrix[0].size();
        bool firstRow = false, firstCol = false;

        for (int j = 0; j < n; j++) {
            if (matrix[0][j] == 0) firstRow = true;
        }
        for (int i = 0; i < m; i++) {
            if (matrix[i][0] == 0) firstCol = true;
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][j] == 0) {
                    matrix[i][0] = 0;
                    matrix[0][j] = 0;
                }
            }
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][0] == 0 || matrix[0][j] == 0) {
                    matrix[i][j] = 0;
                }
            }
        }

        if (firstRow) {
            for (int j = 0; j < n; j++) matrix[0][j] = 0;
        }
        if (firstCol) {
            for (int i = 0; i < m; i++) matrix[i][0] = 0;
        }
    }
};
```

## 2. 螺旋矩阵

链接：https://leetcode.cn/problems/spiral-matrix/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

按顺时针螺旋顺序返回矩阵元素。

### 这题考什么

考察四个边界的维护。

### 生活化理解

像沿房间墙边走一圈，走完一圈后房间向内缩一层。

### 核心思路

维护四个边界：

- `top`
- `bottom`
- `left`
- `right`

每次按右、下、左、上的顺序遍历，并收缩对应边界。

### C++ 代码

```cpp
class Solution {
public:
    vector<int> spiralOrder(vector<vector<int>>& matrix) {
        vector<int> ans;
        int top = 0, bottom = matrix.size() - 1;
        int left = 0, right = matrix[0].size() - 1;

        while (top <= bottom && left <= right) {
            for (int j = left; j <= right; j++) ans.push_back(matrix[top][j]);
            top++;

            for (int i = top; i <= bottom; i++) ans.push_back(matrix[i][right]);
            right--;

            if (top <= bottom) {
                for (int j = right; j >= left; j--) ans.push_back(matrix[bottom][j]);
                bottom--;
            }

            if (left <= right) {
                for (int i = bottom; i >= top; i--) ans.push_back(matrix[i][left]);
                left++;
            }
        }

        return ans;
    }
};
```

## 3. 旋转图像

链接：https://leetcode.cn/problems/rotate-image/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

将 n x n 矩阵原地顺时针旋转 90 度。

### 这题考什么

考察矩阵原地变换。

### 生活化理解

像把一张纸先沿主对角线翻折，再左右翻折。

### 核心思路

顺时针旋转 90 度：

1. 先转置：`matrix[i][j]` 和 `matrix[j][i]` 交换。
2. 再反转每一行。

### C++ 代码

```cpp
class Solution {
public:
    void rotate(vector<vector<int>>& matrix) {
        int n = matrix.size();

        for (int i = 0; i < n; i++) {
            for (int j = i + 1; j < n; j++) {
                swap(matrix[i][j], matrix[j][i]);
            }
        }

        for (int i = 0; i < n; i++) {
            reverse(matrix[i].begin(), matrix[i].end());
        }
    }
};
```

## 4. 搜索二维矩阵 II

链接：https://leetcode.cn/problems/search-a-2d-matrix-ii/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

在行列均递增的矩阵中搜索目标值。

### 这题考什么

考察利用单调性排除行或列。

### 生活化理解

从右上角看，左边都更小，下边都更大，所以每步能排除一行或一列。

### 核心思路

从右上角 `(0, n - 1)` 出发：

- 如果当前值等于目标，返回 true。
- 如果当前值大于目标，往左走。
- 如果当前值小于目标，往下走。

### C++ 代码

```cpp
class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
        int m = matrix.size();
        int n = matrix[0].size();
        int i = 0, j = n - 1;

        while (i < m && j >= 0) {
            if (matrix[i][j] == target) return true;
            if (matrix[i][j] > target) {
                j--;
            } else {
                i++;
            }
        }

        return false;
    }
};
```
