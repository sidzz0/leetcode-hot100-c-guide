# LeetCode Hot 100 - 08 图论 C++ 题解

## 一、图论知识点回顾

Hot100 里的图论不偏竞赛，重点是 DFS/BFS、拓扑排序和 Trie。

常见类型：

- 网格连通块：DFS/BFS。
- 多源扩散：多源 BFS。
- 依赖关系：拓扑排序。
- 前缀匹配：Trie。

方向数组：

```cpp
int dirs[4][2] = {{1,0}, {-1,0}, {0,1}, {0,-1}};
```


## 二、基础容器/语法解释

图论题常用队列、邻接表和方向数组。

### 方向数组

```cpp
int dirs[4][2] = {{1,0}, {-1,0}, {0,1}, {0,-1}};
```

表示向下、向上、向右、向左四个方向。

### queue<pair<int, int>>

```cpp
queue<pair<int, int>> q;
q.push({i, j});
auto [x, y] = q.front();
q.pop();
```

用于 BFS 存储网格坐标。

### 邻接表

```cpp
vector<vector<int>> graph(numCourses);
graph[b].push_back(a);
```

表示从课程 `b` 指向课程 `a`。

### 入度数组

```cpp
vector<int> indegree(numCourses, 0);
indegree[a]++;
```

拓扑排序中用于记录每个节点还有多少前置依赖。

## 1. 岛屿数量

链接：https://leetcode.cn/problems/number-of-islands/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

解决「岛屿数量」对应的图或网格问题。

### 这题考什么

考察 DFS/BFS、拓扑关系或前缀树建模。

### 生活化理解

像在地图中扩散、搜索或处理任务依赖关系。

### 核心思路

遇到一个陆地 `'1'`，答案加一，然后用 DFS/BFS 把与它连通的所有陆地淹掉，避免重复统计。

### C++ 代码

```cpp
class Solution {
public:
    int numIslands(vector<vector<char>>& grid) {
        int m = grid.size();
        int n = grid[0].size();
        int ans = 0;

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == '1') {
                    ans++;
                    dfs(grid, i, j);
                }
            }
        }

        return ans;
    }

    void dfs(vector<vector<char>>& grid, int i, int j) {
        int m = grid.size();
        int n = grid[0].size();
        if (i < 0 || i >= m || j < 0 || j >= n || grid[i][j] != '1') return;

        grid[i][j] = '0';
        dfs(grid, i + 1, j);
        dfs(grid, i - 1, j);
        dfs(grid, i, j + 1);
        dfs(grid, i, j - 1);
    }
};
```

### 复杂度分析

- 时间复杂度：通常为 O(n)，需要访问每个节点或格子。
- 空间复杂度：取决于递归栈、队列、哈希表或堆。

## 2. 腐烂的橘子

链接：https://leetcode.cn/problems/rotting-oranges/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

解决「腐烂的橘子」对应的图或网格问题。

### 这题考什么

考察 DFS/BFS、拓扑关系或前缀树建模。

### 生活化理解

像在地图中扩散、搜索或处理任务依赖关系。

### 核心思路

多源 BFS。初始所有烂橘子同时入队，每一轮让相邻新鲜橘子腐烂。层数就是分钟数。

### C++ 代码

```cpp
class Solution {
public:
    int orangesRotting(vector<vector<int>>& grid) {
        int m = grid.size();
        int n = grid[0].size();
        queue<pair<int, int>> q;
        int fresh = 0;

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == 2) q.push({i, j});
                if (grid[i][j] == 1) fresh++;
            }
        }

        int minutes = 0;
        int dirs[4][2] = {{1,0}, {-1,0}, {0,1}, {0,-1}};

        while (!q.empty() && fresh > 0) {
            int sz = q.size();
            minutes++;

            for (int k = 0; k < sz; k++) {
                auto [x, y] = q.front();
                q.pop();

                for (auto& d : dirs) {
                    int nx = x + d[0], ny = y + d[1];
                    if (nx < 0 || nx >= m || ny < 0 || ny >= n || grid[nx][ny] != 1) continue;
                    grid[nx][ny] = 2;
                    fresh--;
                    q.push({nx, ny});
                }
            }
        }

        return fresh == 0 ? minutes : -1;
    }
};
```

- 时间复杂度：通常为 O(n)，需要访问每个节点或格子。

## 3. 课程表

链接：https://leetcode.cn/problems/course-schedule/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

解决「课程表」对应的图或网格问题。

### 这题考什么

考察 DFS/BFS、拓扑关系或前缀树建模。

### 生活化理解

像在地图中扩散、搜索或处理任务依赖关系。

### 核心思路

判断有向图是否有环。用拓扑排序：

- 建图
- 统计入度
- 入度为 0 的课程入队
- 每学完一门课，降低后继课程入度
- 最后如果学完课程数等于总数，则无环

### C++ 代码

```cpp
class Solution {
public:
    bool canFinish(int numCourses, vector<vector<int>>& prerequisites) {
        vector<vector<int>> graph(numCourses);
        vector<int> indegree(numCourses, 0);

        for (auto& p : prerequisites) {
            int a = p[0], b = p[1];
            graph[b].push_back(a);
            indegree[a]++;
        }

        queue<int> q;
        for (int i = 0; i < numCourses; i++) {
            if (indegree[i] == 0) q.push(i);
        }

        int count = 0;
        while (!q.empty()) {
            int cur = q.front();
            q.pop();
            count++;

            for (int nxt : graph[cur]) {
                indegree[nxt]--;
                if (indegree[nxt] == 0) q.push(nxt);
            }
        }

        return count == numCourses;
    }
};
```

- 时间复杂度：通常为 O(n)，需要访问每个节点或格子。
- 空间复杂度：取决于递归栈、队列、哈希表或堆。

## 4. 实现 Trie 前缀树

链接：https://leetcode.cn/studyplan/top-100-liked/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

解决「实现 Trie 前缀树」对应的图或网格问题。

### 这题考什么

考察 DFS/BFS、拓扑关系或前缀树建模。

### 生活化理解

像在地图中扩散、搜索或处理任务依赖关系。

### 核心思路

Trie 是一棵字符树。每条边代表一个字符，从根到某个节点形成一个前缀。节点里的 `isEnd` 表示是否有单词在这里结束。

### C++ 代码

```cpp
class Trie {
private:
    struct Node {
        vector<Node*> children;
        bool isEnd;
        Node() : children(26, nullptr), isEnd(false) {}
    };

    Node* root;

public:
    Trie() {
        root = new Node();
    }
    
    void insert(string word) {
        Node* cur = root;
        for (char c : word) {
            int idx = c - 'a';
            if (!cur->children[idx]) {
                cur->children[idx] = new Node();
            }
            cur = cur->children[idx];
        }
        cur->isEnd = true;
    }
    
    bool search(string word) {
        Node* node = find(word);
        return node && node->isEnd;
    }
    
    bool startsWith(string prefix) {
        return find(prefix) != nullptr;
    }

private:
    Node* find(const string& s) {
        Node* cur = root;
        for (char c : s) {
            int idx = c - 'a';
            if (!cur->children[idx]) return nullptr;
            cur = cur->children[idx];
        }
        return cur;
    }
};
```
