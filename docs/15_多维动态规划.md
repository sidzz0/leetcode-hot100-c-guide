# LeetCode Hot 100 - 15 多维动态规划 C++ 题解

## 一、多维动态规划知识点回顾

当一个状态需要两个下标描述时，通常使用二维 DP。

常见定义：

- `dp[i][j]`：走到网格 `(i, j)` 的结果。
- `dp[i][j]`：两个字符串前缀之间的关系。
- `dp[i][j]`：区间 `[i, j]` 是否满足某种性质。

二维 DP 最容易错的是初始化和遍历方向。

## 1. 不同路径

链接：https://leetcode.cn/problems/unique-paths/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

用动态规划解决「不同路径」。

### 这题考什么

考察状态定义、状态转移和初始化。

### 生活化理解

像填表格，后面的答案依赖前面已经算好的答案。

### 解题思路

`dp[i][j]` 表示到达 `(i, j)` 的路径数，只能从上方或左方来。

### C++ 代码

```cpp
class Solution {
public:
    int uniquePaths(int m, int n) {
        vector<vector<int>> dp(m, vector<int>(n, 1));

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                dp[i][j] = dp[i - 1][j] + dp[i][j - 1];
            }
        }

        return dp[m - 1][n - 1];
    }
};
```

### 复杂度分析

- 时间复杂度：取决于状态数量和转移复杂度。
- 空间复杂度：取决于 dp 数组维度。

## 2. 最小路径和

链接：https://leetcode.cn/problems/minimum-path-sum/

### 这题考什么

### 生活化理解

像填表格，后面的答案依赖前面已经算好的答案。

### 解题思路

`dp[i][j]` 表示走到 `(i, j)` 的最小路径和，只能从上或左来。

### C++ 代码

```cpp
class Solution {
public:
    int minPathSum(vector<vector<int>>& grid) {
        int m = grid.size();
        int n = grid[0].size();
        vector<vector<int>> dp(m, vector<int>(n, 0));

        dp[0][0] = grid[0][0];
        for (int i = 1; i < m; i++) dp[i][0] = dp[i - 1][0] + grid[i][0];
        for (int j = 1; j < n; j++) dp[0][j] = dp[0][j - 1] + grid[0][j];

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                dp[i][j] = min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j];
            }
        }

        return dp[m - 1][n - 1];
    }
};
```

## 3. 最长回文子串

链接：https://leetcode.cn/problems/longest-palindromic-substring/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

用动态规划解决「最长回文子串」。

### 这题考什么

考察状态定义、状态转移和初始化。

### 生活化理解

像填表格，后面的答案依赖前面已经算好的答案。

### 解题思路

`dp[i][j]` 表示 `s[i..j]` 是否是回文串。

转移：

```text
s[i] == s[j] 且内部也是回文
```

长度小于等于 2 时，只要两端相等就是回文。

### C++ 代码

```cpp
class Solution {
public:
    string longestPalindrome(string s) {
        int n = s.size();
        vector<vector<bool>> dp(n, vector<bool>(n, false));
        int start = 0, maxLen = 1;

        for (int len = 1; len <= n; len++) {
            for (int i = 0; i + len - 1 < n; i++) {
                int j = i + len - 1;
                if (s[i] == s[j] && (len <= 2 || dp[i + 1][j - 1])) {
                    dp[i][j] = true;
                    if (len > maxLen) {
                        start = i;
                        maxLen = len;
                    }
                }
            }
        }

        return s.substr(start, maxLen);
    }
};
```

- 时间复杂度：取决于状态数量和转移复杂度。

### 生活化理解

### 解题思路

`dp[i][j]` 表示 `text1` 前 i 个字符和 `text2` 前 j 个字符的最长公共子序列长度。

### C++ 代码

```cpp
class Solution {
public:
    int longestCommonSubsequence(string text1, string text2) {
        int m = text1.size();
        int n = text2.size();
        vector<vector<int>> dp(m + 1, vector<int>(n + 1, 0));

        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (text1[i - 1] == text2[j - 1]) {
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                } else {
                    dp[i][j] = max(dp[i - 1][j], dp[i][j - 1]);
                }
            }
        }

        return dp[m][n];
    }
};
```

## 5. 编辑距离

链接：https://leetcode.cn/problems/edit-distance/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

### 这题考什么

### 生活化理解

像填表格，后面的答案依赖前面已经算好的答案。

### 解题思路

`dp[i][j]` 表示把 `word1` 前 i 个字符变成 `word2` 前 j 个字符的最少操作数。

如果当前字符相同，不需要操作。否则从插入、删除、替换中取最小。

### C++ 代码

```cpp
class Solution {
public:
    int minDistance(string word1, string word2) {
        int m = word1.size();
        int n = word2.size();
        vector<vector<int>> dp(m + 1, vector<int>(n + 1, 0));

        for (int i = 0; i <= m; i++) dp[i][0] = i;
        for (int j = 0; j <= n; j++) dp[0][j] = j;

        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (word1[i - 1] == word2[j - 1]) {
                    dp[i][j] = dp[i - 1][j - 1];
                } else {
                    dp[i][j] = min({dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1]}) + 1;
                }
            }
        }

        return dp[m][n];
    }
};
```
