## 一、贪心知识点回顾

贪心不是“看起来选最好的”，而是每一步做局部最优，并且局部最优能推出全局最优。

常见贪心信号：

- 维护当前最优状态
- 维护能到达的最远位置
- 当前区间结束时必须做一次选择

## 1. 买卖股票的最佳时机

### 思路

遍历价格，维护历史最低买入价。今天卖出的最大利润就是 `price - minPrice`。

### C++ 代码

```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int minPrice = INT_MAX;
        int ans = 0;

        for (int price : prices) {
            minPrice = min(minPrice, price);
            ans = max(ans, price - minPrice);
        }

        return ans;
    }
};
```

## 2. 跳跃游戏

### 思路

维护当前能到达的最远位置 `far`。遍历到位置 `i` 时，如果 `i > far`，说明到不了这里。否则更新 `far`。

### C++ 代码

```cpp
class Solution {
public:
    bool canJump(vector<int>& nums) {
        int far = 0;

        for (int i = 0; i < nums.size(); i++) {
            if (i > far) return false;
            far = max(far, i + nums[i]);
        }

        return true;
    }
};
```

## 3. 跳跃游戏 II

### 思路

把每次跳跃看成一层 BFS。当前跳跃范围是 `[0, end]`，在这个范围内更新下一次能到达的最远位置 `far`。当遍历到 `end`，必须跳一次。

### C++ 代码

```cpp
class Solution {
public:
    int jump(vector<int>& nums) {
        int steps = 0;
        int end = 0;
        int far = 0;

        for (int i = 0; i < nums.size() - 1; i++) {
            far = max(far, i + nums[i]);
            if (i == end) {
                steps++;
                end = far;
            }
        }

        return steps;
    }
};
```

易错点：循环到 `nums.size() - 2` 即可，避免在终点多跳一次。

## 4. 划分字母区间

### 思路

先记录每个字符最后出现的位置。遍历字符串时维护当前片段最远右边界 `end`。当 `i == end` 时，说明当前片段内所有字符都不会出现在后面，可以切一段。

### C++ 代码

```cpp
class Solution {
public:
    vector<int> partitionLabels(string s) {
        vector<int> last(26, 0);
        for (int i = 0; i < s.size(); i++) {
            last[s[i] - 'a'] = i;
        }

        vector<int> ans;
        int start = 0, end = 0;

        for (int i = 0; i < s.size(); i++) {
            end = max(end, last[s[i] - 'a']);
            if (i == end) {
                ans.push_back(end - start + 1);
                start = i + 1;
            }
        }

        return ans;
    }
};
```
