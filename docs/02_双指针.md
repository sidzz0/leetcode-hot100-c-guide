## 一、双指针知识点回顾

双指针不是一种具体数据结构，而是一种遍历技巧。

核心思想：

> 用两个指针分别负责不同职责，从而减少重复枚举。

常见双指针类型：

| 类型 | 常见场景 |
|---|---|
| 快慢指针 | 链表、原地覆盖、移动元素 |
| 左右指针 | 有序数组、两端收缩、容器问题 |
| 固定一个数 + 双指针 | 三数之和、四数之和 |
| 前后扫描 | 接雨水、前缀/后缀最大值 |

## 二、C++ 基础写法

### 1. 定义两个下标

```cpp
int left = 0;
int right = nums.size() - 1;
```

`left` 从左边开始，`right` 从右边开始。

### 2. 两端向中间移动

```cpp
while (left < right) {
    if (/* 某个条件 */) {
        left++;
    } else {
        right--;
    }
}
```

重点不是这段代码本身，而是：

> 每一步为什么移动左指针或右指针。

### 3. 快慢指针

```cpp
int slow = 0;
for (int fast = 0; fast < nums.size(); fast++) {
    if (/* nums[fast] 是需要保留的元素 */) {
        nums[slow] = nums[fast];
        slow++;
    }
}
```

可以理解为：

- `fast` 负责扫描所有元素
- `slow` 负责记录下一个应该写入的位置

# 三、题目 1：移动零

链接：https://leetcode.cn/problems/move-zeroes/

标签：数组、双指针  
难度：Easy  
重要程度：双指针入门题

## 1. 题意简述

给一个数组，把所有 `0` 移动到数组末尾，同时保持非零元素的相对顺序。

要求原地修改数组。

## 2. 这题考什么

这题考快慢指针：

> `fast` 找非零元素，`slow` 指向下一个非零元素应该放的位置。

## 3. 生活化解释

想象你在整理一排书，中间夹着一些空位。  
你希望把所有真正的书都往前排，空位全部留到最后。

你可以让一个人从左到右检查每个位置：

- 看到书，就放到前面下一个该放书的位置
- 看到空位，就先跳过

最后前面全是书，后面再统一填空位。

## 4. 双指针思路

维护两个指针：

```cpp
int slow = 0;
```

含义：

> `slow` 表示下一个非零元素应该放的位置。

遍历数组：

```cpp
for (int fast = 0; fast < nums.size(); fast++)
```

含义：

> `fast` 负责扫描每一个元素。

如果 `nums[fast] != 0`，说明当前元素需要保留，就把它放到 `slow` 位置：

```cpp
nums[slow] = nums[fast];
slow++;
```

遍历结束后，`[0, slow)` 都是非零元素，剩下的位置填 `0`。

## 5. C++ 代码

```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int slow = 0;

        for (int fast = 0; fast < nums.size(); fast++) {
            if (nums[fast] != 0) {
                nums[slow] = nums[fast];
                slow++;
            }
        }

        for (int i = slow; i < nums.size(); i++) {
            nums[i] = 0;
        }
    }
};
```

## 6. 复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(1)

## 7. 易错点

1. 要保持非零元素相对顺序。
2. 第二轮要从 `slow` 开始填 0。
3. 如果数组本来没有 0，也能正常工作。

## 8. 面试表达

可以这样说：

> 使用快慢指针，fast 扫描数组，slow 记录下一个非零元素应该写入的位置。遇到非零元素就写入 slow 并让 slow 后移。遍历结束后，slow 后面的部分全部置零。

# 四、题目 2：盛最多水的容器

链接：https://leetcode.cn/problems/container-with-most-water/

标签：数组、双指针、贪心  
难度：Medium  
重要程度：双指针经典题

## 1. 题意简述

给一组高度，选择两条线作为容器左右边界，使容器能装最多水。

容积由两部分决定：

```text
宽度 * 较短边高度
```

## 2. 这题考什么

这题考左右指针和贪心选择：

> 每次移动较短的那条边。

## 3. 生活化解释

想象你用两块木板夹出一个水槽。  
水槽能装多少水，取决于：

- 两块木板之间有多宽
- 较矮的那块木板有多高

如果左边木板更矮，你移动右边更高的木板，水位还是被左边矮木板限制，而且宽度还变小了。  
所以想变好，只能尝试换掉较矮的那块板。

## 4. 双指针思路

初始化：

```cpp
int left = 0;
int right = height.size() - 1;
```

每一步计算当前面积：

```cpp
int area = min(height[left], height[right]) * (right - left);
```

然后移动较短的一边：

```cpp
if (height[left] < height[right]) {
    left++;
} else {
    right--;
}
```

为什么移动短边？

因为面积高度由短边决定。  
如果移动长边，宽度变小，高度上限仍然被短边卡住，不可能变得更优。

## 5. C++ 代码

```cpp
class Solution {
public:
    int maxArea(vector<int>& height) {
        int left = 0;
        int right = height.size() - 1;
        int ans = 0;

        while (left < right) {
            int h = min(height[left], height[right]);
            int w = right - left;
            ans = max(ans, h * w);

            if (height[left] < height[right]) {
                left++;
            } else {
                right--;
            }
        }

        return ans;
    }
};
```

## 6. 复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(1)

## 7. 易错点

1. 面积高度取两边较小值。
2. 每次移动较短边。
3. `right - left` 是宽度，不是 `right - left + 1`。

## 8. 面试表达

可以这样说：

> 从最宽的左右边界开始计算面积。由于面积受较短边限制，如果移动较长边，宽度变小而高度上限不变，不可能得到更大面积。因此每次移动较短边，尝试找到更高的边界。

# 五、题目 3：三数之和

链接：https://leetcode.cn/problems/3sum/

标签：数组、排序、双指针  
难度：Medium  
重要程度：高频面试题

## 1. 题意简述

给一个整数数组，找出所有不重复的三元组，使三数之和等于 0。

## 2. 这题考什么

这题考的是：

> 排序后固定一个数，把三数之和转化成两数之和。

## 3. 生活化解释

你要凑三个人的账单，使总和为 0。  
先固定第一个人，比如他花了 `-4`，剩下两个人就要凑 `4`。

如果账单已经按金额排好，就可以让一个人从左边找小数，另一个人从右边找大数：

- 和太小，左边往右走，让总和变大
- 和太大，右边往左走，让总和变小

## 4. 双指针思路

先排序：

```cpp
sort(nums.begin(), nums.end());
```

然后枚举第一个数 `nums[i]`。

固定 `i` 后，问题变成：

```text
在 i 右边找两个数，使 nums[left] + nums[right] == -nums[i]
```

左右指针：

```cpp
int left = i + 1;
int right = nums.size() - 1;
```

判断三数和：

- 如果 `sum == 0`，记录答案，并跳过重复值
- 如果 `sum < 0`，说明太小，`left++`
- 如果 `sum > 0`，说明太大，`right--`

## 5. 去重逻辑

三数之和最容易错在去重。

### 固定数去重

```cpp
if (i > 0 && nums[i] == nums[i - 1]) continue;
```

意思是：

> 如果当前固定的数和上一次一样，就会产生重复答案，跳过。

### left / right 去重

找到一个答案后：

```cpp
while (left < right && nums[left] == nums[left + 1]) left++;
while (left < right && nums[right] == nums[right - 1]) right--;
```

然后再移动：

```cpp
left++;
right--;
```

## 6. C++ 代码

```cpp
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        vector<vector<int>> ans;
        sort(nums.begin(), nums.end());

        for (int i = 0; i < nums.size(); i++) {
            if (i > 0 && nums[i] == nums[i - 1]) continue;
            if (nums[i] > 0) break;

            int left = i + 1;
            int right = nums.size() - 1;

            while (left < right) {
                int sum = nums[i] + nums[left] + nums[right];

                if (sum == 0) {
                    ans.push_back({nums[i], nums[left], nums[right]});

                    while (left < right && nums[left] == nums[left + 1]) left++;
                    while (left < right && nums[right] == nums[right - 1]) right--;

                    left++;
                    right--;
                } else if (sum < 0) {
                    left++;
                } else {
                    right--;
                }
            }
        }

        return ans;
    }
};
```

## 7. 复杂度

- 时间复杂度：O(n^2)
- 空间复杂度：O(1)，不计返回结果

## 8. 易错点

1. 必须先排序。
2. 固定数 `i` 要去重。
3. 找到答案后 `left` 和 `right` 都要去重。
4. `nums[i] > 0` 时可以提前结束，因为后面都更大。

## 9. 面试表达

可以这样说：

> 先排序。枚举第一个数，然后在它右侧用双指针寻找另外两个数。由于数组有序，三数和小于 0 时左指针右移，大于 0 时右指针左移。为了避免重复答案，需要对固定数和左右指针都做去重。

# 六、题目 4：接雨水

链接：https://leetcode.cn/problems/trapping-rain-water/

标签：数组、双指针、动态规划、单调栈  
难度：Hard  
重要程度：高频经典难题

## 1. 题意简述

给一组柱子高度，计算下雨后这些柱子之间最多能接多少水。

## 2. 这题考什么

每个位置能接多少水，取决于：

```text
min(左边最高柱子, 右边最高柱子) - 当前柱子高度
```

如果这个值大于 0，就能接水。

## 3. 生活化解释

想象一排高低不同的挡板。  
某个位置能存多少水，不看它自己有多高，而看它左边最高挡板和右边最高挡板中较矮的那个。

因为水会从较矮的一边流出去。

## 4. 方法一：前后缀最大值

先预处理两个数组：

```cpp
leftMax[i] = i 左边包括 i 的最高柱子
rightMax[i] = i 右边包括 i 的最高柱子
```

则位置 `i` 的接水量：

```cpp
min(leftMax[i], rightMax[i]) - height[i]
```

## 5. C++ 代码：前后缀法

```cpp
class Solution {
public:
    int trap(vector<int>& height) {
        int n = height.size();
        if (n == 0) return 0;

        vector<int> leftMax(n), rightMax(n);
        leftMax[0] = height[0];
        rightMax[n - 1] = height[n - 1];

        for (int i = 1; i < n; i++) {
            leftMax[i] = max(leftMax[i - 1], height[i]);
        }

        for (int i = n - 2; i >= 0; i--) {
            rightMax[i] = max(rightMax[i + 1], height[i]);
        }

        int ans = 0;
        for (int i = 0; i < n; i++) {
            ans += min(leftMax[i], rightMax[i]) - height[i];
        }

        return ans;
    }
};
```

## 6. 方法二：双指针优化

前后缀法用了两个数组。  
双指针可以把空间优化到 O(1)。

维护：

```cpp
int leftMax = 0;
int rightMax = 0;
```

当 `height[left] < height[right]` 时，左边的接水量由 `leftMax` 决定。  
因为右边已经有一个更高的边界，短板在左边。

## 7. C++ 代码：双指针法

```cpp
class Solution {
public:
    int trap(vector<int>& height) {
        int left = 0;
        int right = height.size() - 1;
        int leftMax = 0;
        int rightMax = 0;
        int ans = 0;

        while (left < right) {
            leftMax = max(leftMax, height[left]);
            rightMax = max(rightMax, height[right]);

            if (height[left] < height[right]) {
                ans += leftMax - height[left];
                left++;
            } else {
                ans += rightMax - height[right];
                right--;
            }
        }

        return ans;
    }
};
```

## 8. 复杂度

前后缀法：

- 时间复杂度：O(n)
- 空间复杂度：O(n)

双指针法：

- 时间复杂度：O(n)
- 空间复杂度：O(1)

## 9. 易错点

1. 每个位置接水量由左右最高柱子的较小值决定。
2. 前后缀数组要包含当前位置。
3. 双指针法里要先更新 `leftMax` / `rightMax`。
4. `leftMax - height[left]` 不会为负，因为已经更新过 `leftMax`。

## 10. 面试表达

可以这样说：

> 对每个位置，它能接的水等于左右最高柱子中较小者减去当前位置高度。最直观的方法是预处理 leftMax 和 rightMax。进一步可以用双指针优化空间，维护左右两边当前最大值，每次移动较低的一侧，因为这一侧的水位已经可以确定。

# 七、本章总结

## 1. 四道题对应四种双指针能力

| 题目 | 双指针类型 | 核心 |
|---|---|---|
| 移动零 | 快慢指针 | fast 扫描，slow 写入 |
| 盛最多水的容器 | 左右指针 | 移动较短边 |

| 接雨水 | 左右指针 / 前后缀 | 水位由左右最大值较小者决定 |

- 能区分双指针和滑动窗口
