## 一、二叉树知识点回顾

二叉树题优先思考递归。递归三件事：

1. 当前节点要做什么？
2. 左右子树返回什么？
3. 空节点怎么处理？

常见遍历：

- 前序：根左右，适合自顶向下传递信息
- 中序：左根右，BST 中序有序
- 后序：左右根，适合从子树收集信息
- 层序：BFS 队列

LeetCode 常用结构：

```cpp
struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode() : val(0), left(nullptr), right(nullptr) {}
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
    TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
};
```

## 1. 二叉树的中序遍历

### 思路

中序遍历顺序是左、根、右。

### C++ 代码

```cpp
class Solution {
public:
    vector<int> inorderTraversal(TreeNode* root) {
        vector<int> ans;
        dfs(root, ans);
        return ans;
    }

    void dfs(TreeNode* root, vector<int>& ans) {
        if (!root) return;
        dfs(root->left, ans);
        ans.push_back(root->val);
        dfs(root->right, ans);
    }
};
```

## 2. 二叉树的最大深度

### 思路

当前树最大深度 = 左右子树最大深度 + 1。

### C++ 代码

```cpp
class Solution {
public:
    int maxDepth(TreeNode* root) {
        if (!root) return 0;
        return max(maxDepth(root->left), maxDepth(root->right)) + 1;
    }
};
```

## 3. 翻转二叉树

### 思路

交换每个节点的左右子树，然后递归处理左右子树。

### C++ 代码

```cpp
class Solution {
public:
    TreeNode* invertTree(TreeNode* root) {
        if (!root) return nullptr;
        swap(root->left, root->right);
        invertTree(root->left);
        invertTree(root->right);
        return root;
    }
};
```

## 4. 对称二叉树

### 思路

判断左右子树是否镜像：

- 左节点值等于右节点值
- 左的左和右的右对称
- 左的右和右的左对称

### C++ 代码

```cpp
class Solution {
public:
    bool isSymmetric(TreeNode* root) {
        return check(root->left, root->right);
    }

    bool check(TreeNode* a, TreeNode* b) {
        if (!a && !b) return true;
        if (!a || !b) return false;
        return a->val == b->val &&
               check(a->left, b->right) &&
               check(a->right, b->left);
    }
};
```

## 5. 二叉树的直径

### 思路

直径可能经过某个节点，长度是左子树深度 + 右子树深度。后序遍历时计算每个节点深度并更新答案。

### C++ 代码

```cpp
class Solution {
public:
    int ans = 0;

    int diameterOfBinaryTree(TreeNode* root) {
        depth(root);
        return ans;
    }

    int depth(TreeNode* root) {
        if (!root) return 0;
        int left = depth(root->left);
        int right = depth(root->right);
        ans = max(ans, left + right);
        return max(left, right) + 1;
    }
};
```

## 6. 二叉树的层序遍历

### 思路

用队列 BFS，每次处理一整层。

### C++ 代码

```cpp
class Solution {
public:
    vector<vector<int>> levelOrder(TreeNode* root) {
        vector<vector<int>> ans;
        if (!root) return ans;

        queue<TreeNode*> q;
        q.push(root);

        while (!q.empty()) {
            int sz = q.size();
            vector<int> level;

            for (int i = 0; i < sz; i++) {
                TreeNode* node = q.front();
                q.pop();
                level.push_back(node->val);
                if (node->left) q.push(node->left);
                if (node->right) q.push(node->right);
            }

            ans.push_back(level);
        }

        return ans;
    }
};
```

## 7. 将有序数组转换为二叉搜索树

### 思路

每次选择中点作为根，左半边构造左子树，右半边构造右子树。

### C++ 代码

```cpp
class Solution {
public:
    TreeNode* sortedArrayToBST(vector<int>& nums) {
        return build(nums, 0, nums.size() - 1);
    }

    TreeNode* build(vector<int>& nums, int l, int r) {
        if (l > r) return nullptr;
        int mid = l + (r - l) / 2;
        TreeNode* root = new TreeNode(nums[mid]);
        root->left = build(nums, l, mid - 1);
        root->right = build(nums, mid + 1, r);
        return root;
    }
};
```

## 8. 验证二叉搜索树

### 思路

BST 中序遍历必须严格递增。

### C++ 代码

```cpp
class Solution {
public:
    long long prev = LLONG_MIN;

    bool isValidBST(TreeNode* root) {
        if (!root) return true;
        if (!isValidBST(root->left)) return false;
        if (root->val <= prev) return false;
        prev = root->val;
        return isValidBST(root->right);
    }
};
```

易错点：不能只比较父子节点，必须保证整棵左子树都小于根、整棵右子树都大于根。

## 9. 二叉搜索树中第 K 小的元素

### 思路

BST 中序遍历是升序，第 k 个访问到的节点就是答案。

### C++ 代码

```cpp
class Solution {
public:
    int count = 0;
    int ans = 0;

    int kthSmallest(TreeNode* root, int k) {
        dfs(root, k);
        return ans;
    }

    void dfs(TreeNode* root, int k) {
        if (!root) return;
        dfs(root->left, k);
        count++;
        if (count == k) {
            ans = root->val;
            return;
        }
        dfs(root->right, k);
    }
};
```

## 10. 二叉树的右视图

### 思路

层序遍历，每层最后一个节点就是右视图。

### C++ 代码

```cpp
class Solution {
public:
    vector<int> rightSideView(TreeNode* root) {
        vector<int> ans;
        if (!root) return ans;

        queue<TreeNode*> q;
        q.push(root);

        while (!q.empty()) {
            int sz = q.size();
            for (int i = 0; i < sz; i++) {
                TreeNode* node = q.front();
                q.pop();
                if (i == sz - 1) ans.push_back(node->val);
                if (node->left) q.push(node->left);
                if (node->right) q.push(node->right);
            }
        }

        return ans;
    }
};
```

## 11. 二叉树展开为链表

### 思路

后序处理。先把左右子树展开，然后把左子树接到右边，再把原右子树接到新右链末尾。

### C++ 代码

```cpp
class Solution {
public:
    void flatten(TreeNode* root) {
        if (!root) return;

        flatten(root->left);
        flatten(root->right);

        TreeNode* left = root->left;
        TreeNode* right = root->right;

        root->left = nullptr;
        root->right = left;

        TreeNode* cur = root;
        while (cur->right) cur = cur->right;
        cur->right = right;
    }
};
```

## 12. 从前序与中序遍历序列构造二叉树

### 思路

前序第一个元素是根。用哈希表记录中序位置，快速确定左右子树范围。

### C++ 代码

```cpp
class Solution {
public:
    unordered_map<int, int> pos;

    TreeNode* buildTree(vector<int>& preorder, vector<int>& inorder) {
        for (int i = 0; i < inorder.size(); i++) {
            pos[inorder[i]] = i;
        }
        return build(preorder, 0, preorder.size() - 1, 0, inorder.size() - 1);
    }

    TreeNode* build(vector<int>& preorder, int pl, int pr, int il, int ir) {
        if (pl > pr) return nullptr;

        int rootVal = preorder[pl];
        int k = pos[rootVal];
        int leftSize = k - il;

        TreeNode* root = new TreeNode(rootVal);
        root->left = build(preorder, pl + 1, pl + leftSize, il, k - 1);
        root->right = build(preorder, pl + leftSize + 1, pr, k + 1, ir);
        return root;
    }
};
```

## 13. 路径总和 III

### 思路

前缀和 + 哈希表。当前路径前缀和为 `sum`，如果之前出现过 `sum - targetSum`，说明中间这段路径和为目标值。

### C++ 代码

```cpp
class Solution {
public:
    unordered_map<long long, int> cnt;
    int ans = 0;

    int pathSum(TreeNode* root, int targetSum) {
        cnt[0] = 1;
        dfs(root, 0, targetSum);
        return ans;
    }

    void dfs(TreeNode* root, long long sum, int target) {
        if (!root) return;

        sum += root->val;
        if (cnt.count(sum - target)) ans += cnt[sum - target];

        cnt[sum]++;
        dfs(root->left, sum, target);
        dfs(root->right, sum, target);
        cnt[sum]--;
    }
};
```

易错点：回溯时要 `cnt[sum]--`，否则会影响其他路径。

## 14. 二叉树的最近公共祖先

### 思路

如果当前节点是空、p、q，直接返回当前节点。  
分别在左右子树找 p/q：

- 左右都找到，当前节点就是最近公共祖先。
- 只找到一边，返回那一边。

### C++ 代码

```cpp
class Solution {
public:
    TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {
        if (!root || root == p || root == q) return root;

        TreeNode* left = lowestCommonAncestor(root->left, p, q);
        TreeNode* right = lowestCommonAncestor(root->right, p, q);

        if (left && right) return root;
        return left ? left : right;
    }
};
```

## 15. 二叉树中的最大路径和

### 思路

每个节点向父节点最多只能贡献一条路径，所以返回值是：

```text
节点值 + max(左贡献, 右贡献, 0)
```

但全局答案可以经过当前节点并同时连接左右子树：

```text
节点值 + 左贡献 + 右贡献
```

### C++ 代码

```cpp
class Solution {
public:
    int ans = INT_MIN;

    int maxPathSum(TreeNode* root) {
        gain(root);
        return ans;
    }

    int gain(TreeNode* root) {
        if (!root) return 0;

        int left = max(0, gain(root->left));
        int right = max(0, gain(root->right));

        ans = max(ans, root->val + left + right);

        return root->val + max(left, right);
    }
};
```

易错点：返回给父节点的路径不能同时走左右两边。
