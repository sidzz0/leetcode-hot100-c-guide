# LeetCode Hot 100 - 07 二叉树 C++ 题解

## 一、二叉树知识点回顾

二叉树题优先从递归想。递归不是玄学，而是把问题交给左右子树，然后用左右子树的结果处理当前节点。

递归三问：

1. 当前节点要做什么？
2. 左右子树要返回什么？
3. 空节点怎么处理？

遍历类型：

- 前序：根左右，适合自顶向下传信息。
- 中序：左根右，BST 中序有序。
- 后序：左右根，适合从子树收集信息。
- 层序：BFS 队列。


## 二、基础容器/语法解释

二叉树题主要使用递归和队列。

### TreeNode 结构

```cpp
struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
};
```

- `val`：当前节点值。
- `left`：左子树。
- `right`：右子树。

### 递归空节点判断

```cpp
if (!root) return;
```

表示当前节点为空，递归结束。

### queue 层序遍历

```cpp
queue<TreeNode*> q;
q.push(root);
q.front();
q.pop();
```

- `push`：入队。
- `front`：取队头。
- `pop`：弹出队头。

### unordered_map 存位置

```cpp
unordered_map<int, int> pos;
pos[inorder[i]] = i;
```

构造二叉树时，用它快速查找某个值在中序遍历中的位置。

### 核心思路

中序遍历顺序是左、根、右。

### C++ 代码

```cpp
class Solution {
public:
    vector<int> inorderTraversal(TreeNode* root) {
        vector<int> ans;
        dfs(root, ans);
        return ans;
    }

    void dfs(TreeNode* root, vector<int>& ans) {
        if (!root) return;
        dfs(root->left, ans);
        ans.push_back(root->val);
        dfs(root->right, ans);
    }
};
```

```cpp
               check(a->left, b->right) &&

};
```

```cpp
class Solution {
public:
    int ans = 0;

    int diameterOfBinaryTree(TreeNode* root) {
        depth(root);
        return ans;
    }

        if (!root) return 0;
        int left = depth(root->left);
        int right = depth(root->right);
        ans = max(ans, left + right);
        return max(left, right) + 1;
    }
};
```

### C++ 代码

```cpp
class Solution {
public:
    vector<vector<int>> levelOrder(TreeNode* root) {
        vector<vector<int>> ans;
        if (!root) return ans;

        queue<TreeNode*> q;
        q.push(root);

        while (!q.empty()) {
            int sz = q.size();
            vector<int> level;

            for (int i = 0; i < sz; i++) {
                TreeNode* node = q.front();
                q.pop();
                level.push_back(node->val);
                if (node->left) q.push(node->left);
                if (node->right) q.push(node->right);
            }

            ans.push_back(level);
        }

        return ans;
    }
};
```

```cpp
class Solution {
public:
    unordered_map<int, int> pos;

    TreeNode* buildTree(vector<int>& preorder, vector<int>& inorder) {
        for (int i = 0; i < inorder.size(); i++) {
            pos[inorder[i]] = i;
        }
        return build(preorder, 0, preorder.size() - 1, 0, inorder.size() - 1);
    }

    TreeNode* build(vector<int>& preorder, int pl, int pr, int il, int ir) {
        if (pl > pr) return nullptr;

        int rootVal = preorder[pl];
        int k = pos[rootVal];
        int leftSize = k - il;

        TreeNode* root = new TreeNode(rootVal);
        root->left = build(preorder, pl + 1, pl + leftSize, il, k - 1);
        root->right = build(preorder, pl + leftSize + 1, pr, k + 1, ir);
        return root;
    }
};
```

### C++ 代码

```cpp
class Solution {
public:
    unordered_map<long long, int> cnt;
    int ans = 0;

    int pathSum(TreeNode* root, int targetSum) {
        cnt[0] = 1;
        dfs(root, 0, targetSum);
        return ans;
    }

    void dfs(TreeNode* root, long long sum, int target) {
        if (!root) return;

        sum += root->val;
        if (cnt.count(sum - target)) ans += cnt[sum - target];

        cnt[sum]++;
        dfs(root->left, sum, target);
        dfs(root->right, sum, target);
        cnt[sum]--;
    }
};
```

```cpp
class Solution {
public:
    TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {

        TreeNode* right = lowestCommonAncestor(root->right, p, q);

        if (left && right) return root;

};
```

### 题意简述

### 核心思路

每个节点向父节点最多只能贡献一条路径，所以返回值是：

```text
节点值 + max(左贡献, 右贡献, 0)
```

但全局答案可以经过当前节点并同时连接左右子树：

```text
节点值 + 左贡献 + 右贡献
```

### C++ 代码

```cpp
class Solution {
public:
    int ans = INT_MIN;

    int maxPathSum(TreeNode* root) {
        gain(root);
        return ans;
    }

    int gain(TreeNode* root) {
        if (!root) return 0;

        int left = max(0, gain(root->left));
        int right = max(0, gain(root->right));

        ans = max(ans, root->val + left + right);

        return root->val + max(left, right);
    }
};
```

- 时间复杂度：通常为 O(n)，需要访问每个节点或格子。
