# LeetCode Hot 100 - 10 二分查找 C++ 题解

## 一、二分查找知识点回顾

二分查找的核心不是 `mid` 怎么写，而是每次能排除一半搜索空间。

你需要明确：

- 搜索区间是什么？
- `mid` 满足条件时，答案在左边还是右边？
- 最后返回 `left`、`right` 还是直接返回 `mid`？

常用左闭右开 lower_bound 模板：

```cpp
int l = 0, r = nums.size();
while (l < r) {
    int mid = l + (r - l) / 2;
    if (nums[mid] < target) l = mid + 1;
    else r = mid;
}
return l;
```

## 1. 搜索插入位置

链接：https://leetcode.cn/problems/search-insert-position/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

在有序或可二分的搜索空间中解决「搜索插入位置」。

### 这题考什么

考察搜索空间收缩和边界处理。

### 生活化理解

像猜数字，每次根据中间结果排除一半可能。

### 解题思路

找第一个大于等于 `target` 的位置。

### C++ 代码

```cpp
class Solution {
public:
    int searchInsert(vector<int>& nums, int target) {
        int left = 0, right = nums.size();

        while (left < right) {
            int mid = left + (right - left) / 2;
            if (nums[mid] < target) {
                left = mid + 1;
            } else {
                right = mid;
            }
        }

        return left;
    }
};
```

### 复杂度分析

- 时间复杂度：通常为 O(log n)，个别题为 O(log(m+n)) 或 O(n log n)。
- 空间复杂度：通常为 O(1)，递归写法除外。

- 先明确题目模型，不要直接套模板。
- 注意空输入、边界下标和重复元素。

## 3. 查找元素的第一个和最后一个位置

在有序或可二分的搜索空间中解决「查找元素的第一个和最后一个位置」。

### 这题考什么

### 生活化理解

像猜数字，每次根据中间结果排除一半可能。

### 解题思路

分别找：

- 第一个大于等于 target 的位置
- 第一个大于 target 的位置，再减一

### C++ 代码

```cpp
class Solution {
public:
    vector<int> searchRange(vector<int>& nums, int target) {
        int left = lowerBound(nums, target);
        int right = lowerBound(nums, target + 1) - 1;

        if (left <= right && right < nums.size() && nums[left] == target) {
            return {left, right};
        }
        return {-1, -1};
    }

    int lowerBound(vector<int>& nums, long long target) {
        int l = 0, r = nums.size();
        while (l < r) {
            int mid = l + (r - l) / 2;
            if (nums[mid] < target) l = mid + 1;
            else r = mid;
        }
        return l;
    }
};
```

## 4. 搜索旋转排序数组

链接：https://leetcode.cn/problems/search-in-rotated-sorted-array/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

在有序或可二分的搜索空间中解决「搜索旋转排序数组」。

### 这题考什么

考察搜索空间收缩和边界处理。

### 生活化理解

像猜数字，每次根据中间结果排除一半可能。

### 解题思路

每次二分后，左右两边至少有一边是有序的。判断 target 是否落在有序区间里。

### C++ 代码

```cpp
class Solution {
public:
    int search(vector<int>& nums, int target) {
        int left = 0, right = nums.size() - 1;

        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (nums[mid] == target) return mid;

            if (nums[left] <= nums[mid]) {
                if (nums[left] <= target && target < nums[mid]) right = mid - 1;
                else left = mid + 1;
            } else {
                if (nums[mid] < target && target <= nums[right]) left = mid + 1;
                else right = mid - 1;
            }
        }

        return -1;
    }
};
```

### 复杂度分析

- 时间复杂度：通常为 O(log n)，个别题为 O(log(m+n)) 或 O(n log n)。

### 题意简述

考察搜索空间收缩和边界处理。

### 生活化理解

像猜数字，每次根据中间结果排除一半可能。

### 解题思路

和右端点比较：

- 如果 `nums[mid] > nums[right]`，最小值在右边。
- 否则最小值在左边或 mid。

### C++ 代码

```cpp
class Solution {
public:
    int findMin(vector<int>& nums) {
        int left = 0, right = nums.size() - 1;

        while (left < right) {
            int mid = left + (right - left) / 2;
            if (nums[mid] > nums[right]) {
                left = mid + 1;
            } else {
                right = mid;
            }
        }

        return nums[left];
    }
};
```

## 6. 寻找两个正序数组的中位数

链接：https://leetcode.cn/problems/median-of-two-sorted-arrays/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

在有序或可二分的搜索空间中解决「寻找两个正序数组的中位数」。

### 这题考什么

考察搜索空间收缩和边界处理。

### 生活化理解

像猜数字，每次根据中间结果排除一半可能。

### 解题思路

把问题转成找两个有序数组中的第 k 小元素。每次比较两个数组第 `k/2` 个元素，排除较小那一边的前 `k/2` 个元素。

### C++ 代码

```cpp
class Solution {
public:
    double findMedianSortedArrays(vector<int>& nums1, vector<int>& nums2) {
        int total = nums1.size() + nums2.size();
        if (total % 2 == 1) {
            return getKth(nums1, 0, nums2, 0, total / 2 + 1);
        }
        int left = getKth(nums1, 0, nums2, 0, total / 2);
        int right = getKth(nums1, 0, nums2, 0, total / 2 + 1);
        return (left + right) / 2.0;
    }

    int getKth(vector<int>& a, int i, vector<int>& b, int j, int k) {
        if (i >= a.size()) return b[j + k - 1];
        if (j >= b.size()) return a[i + k - 1];
        if (k == 1) return min(a[i], b[j]);

        int half = k / 2;
        int newI = min(i + half, (int)a.size()) - 1;
        int newJ = min(j + half, (int)b.size()) - 1;

        if (a[newI] <= b[newJ]) {
            return getKth(a, newI + 1, b, j, k - (newI - i + 1));
        } else {
            return getKth(a, i, b, newJ + 1, k - (newJ - j + 1));
        }
    }
};
```
