## 一、栈知识点回顾

栈适合处理“最近的还没解决的问题”。

常用操作：

```cpp
stack<int> st;
st.push(x);
st.top();
st.pop();
st.empty();
```

单调栈用于找下一个更大/更小元素。

## 1. 有效的括号

### 思路

左括号入栈，遇到右括号时检查栈顶是否匹配。

### C++ 代码

```cpp
class Solution {
public:
    bool isValid(string s) {
        stack<char> st;

        for (char c : s) {
            if (c == '(' || c == '[' || c == '{') {
                st.push(c);
            } else {
                if (st.empty()) return false;
                char t = st.top();
                st.pop();
                if ((c == ')' && t != '(') ||
                    (c == ']' && t != '[') ||
                    (c == '}' && t != '{')) {
                    return false;
                }
            }
        }

        return st.empty();
    }
};
```

## 2. 最小栈

### 思路

一个栈存所有元素，另一个栈同步存当前最小值。

### C++ 代码

```cpp
class MinStack {
private:
    stack<int> data;
    stack<int> mins;

public:
    MinStack() {}
    
    void push(int val) {
        data.push(val);
        if (mins.empty()) mins.push(val);
        else mins.push(min(val, mins.top()));
    }
    
    void pop() {
        data.pop();
        mins.pop();
    }
    
    int top() {
        return data.top();
    }
    
    int getMin() {
        return mins.top();
    }
};
```

## 3. 字符串解码

### 思路

遇到 `[` 时保存当前字符串和重复次数；遇到 `]` 时弹出上一层状态并拼接。

### C++ 代码

```cpp
class Solution {
public:
    string decodeString(string s) {
        stack<int> nums;
        stack<string> strs;
        string cur;
        int num = 0;

        for (char c : s) {
            if (isdigit(c)) {
                num = num * 10 + (c - '0');
            } else if (c == '[') {
                nums.push(num);
                strs.push(cur);
                num = 0;
                cur.clear();
            } else if (c == ']') {
                int times = nums.top();
                nums.pop();
                string prev = strs.top();
                strs.pop();
                while (times--) prev += cur;
                cur = prev;
            } else {
                cur.push_back(c);
            }
        }

        return cur;
    }
};
```

## 4. 每日温度

### 思路

单调递减栈，存下标。当前温度比栈顶高时，说明找到了栈顶那天的下一个更高温。

### C++ 代码

```cpp
class Solution {
public:
    vector<int> dailyTemperatures(vector<int>& temperatures) {
        int n = temperatures.size();
        vector<int> ans(n, 0);
        stack<int> st;

        for (int i = 0; i < n; i++) {
            while (!st.empty() && temperatures[i] > temperatures[st.top()]) {
                int idx = st.top();
                st.pop();
                ans[idx] = i - idx;
            }
            st.push(i);
        }

        return ans;
    }
};
```

## 5. 柱状图中最大的矩形

### 思路

单调递增栈。当当前柱子低于栈顶柱子时，说明栈顶柱子的右边界确定了；左边界是弹出后新的栈顶。

在左右两端加高度 0 的哨兵，方便统一处理。

### C++ 代码

```cpp
class Solution {
public:
    int largestRectangleArea(vector<int>& heights) {
        vector<int> h;
        h.push_back(0);
        for (int x : heights) h.push_back(x);
        h.push_back(0);

        stack<int> st;
        int ans = 0;

        for (int i = 0; i < h.size(); i++) {
            while (!st.empty() && h[i] < h[st.top()]) {
                int height = h[st.top()];
                st.pop();
                int width = i - st.top() - 1;
                ans = max(ans, height * width);
            }
            st.push(i);
        }

        return ans;
    }
};
```
