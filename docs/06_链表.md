# LeetCode Hot 100 - 06 链表 C++ 题解

## 一、链表知识点回顾

链表题的本质是“指针重连”。数组换位置像搬座位，链表换位置像重新牵手。做链表题最怕的不是思路，而是断链。

常用技巧：

- `dummy` 虚拟头节点：头节点可能变化时使用。
- 快慢指针：找中点、找环、倒数第 N 个节点。
- 反转链表：几乎所有链表进阶题的基本功。
- 哈希表：复制随机链表、记录节点。
- 分治/堆：合并 K 个链表。

LeetCode 常用结构：

```cpp
struct ListNode {
    int val;
    ListNode *next;
    ListNode() : val(0), next(nullptr) {}
    ListNode(int x) : val(x), next(nullptr) {}
    ListNode(int x, ListNode *next) : val(x), next(next) {}
};
```


## 二、基础容器/语法解释

链表题最重要的是指针。

### ListNode 结构

```cpp
struct ListNode {
    int val;
    ListNode *next;
    ListNode() : val(0), next(nullptr) {}
    ListNode(int x) : val(x), next(nullptr) {}
    ListNode(int x, ListNode *next) : val(x), next(next) {}
};
```

- `val`：当前节点的值。
- `next`：指向下一个节点。
- `nullptr`：空指针，表示没有下一个节点。

### 指针访问

```cpp
cur->val;
cur->next;
```

`cur->next` 等价于 `(*cur).next`，表示访问指针指向节点的成员。

### dummy 虚拟头节点

```cpp
ListNode dummy(0, head);
ListNode* cur = &dummy;
return dummy.next;
```

当头节点可能变化时，用 `dummy` 可以统一处理删除、交换、合并等操作。

### new 创建节点

```cpp
cur->next = new ListNode(sum % 10);
```

创建一个新链表节点，并接到当前链表后面。

## 1. 相交链表

链接：https://leetcode.cn/problems/intersection-of-two-linked-lists/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

判断两条链表是否相交，并返回交点。

### 这题考什么

考察双指针消除链表长度差。

### 生活化理解

两个人走不同长度的路，走完自己的路后再走对方的路，总路程就一样了。

### 核心思路

两个指针分别从 A、B 出发。走到尾部后切换到另一条链表头。若相交，二者会在交点相遇；若不相交，最后同时为 `nullptr`。

像两个人走两条不同长度的路，走完自己的路后再走对方的路，总路程相同。

### C++ 代码

```cpp
class Solution {
public:
    ListNode *getIntersectionNode(ListNode *headA, ListNode *headB) {
        ListNode* p = headA;
        ListNode* q = headB;

        while (p != q) {
            p = p ? p->next : headB;
            q = q ? q->next : headA;
        }

        return p;
    }
};
```

### 复杂度分析

- 时间复杂度：通常为 O(n)，需要访问每个节点或格子。
- 空间复杂度：取决于递归栈、队列、哈希表或堆。

### 核心思路

```cpp
            cur->next = prev;
            prev = cur;
            cur = nxt;
        }

        return prev;
    }
};
```

## 3. 回文链表

链接：https://leetcode.cn/problems/palindrome-linked-list/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

判断链表是否为回文。

### 这题考什么

考察快慢指针找中点和反转后半段。

### 生活化理解

像检查一串珠子是否左右对称，先找到中点，再把后半串翻过来比较。

### 核心思路

用快慢指针找到中点，反转后半段，然后比较前半段和后半段。

### C++ 代码

```cpp
class Solution {
public:
    bool isPalindrome(ListNode* head) {
        ListNode* slow = head;
        ListNode* fast = head;

        while (fast && fast->next) {
            slow = slow->next;
            fast = fast->next->next;
        }

        ListNode* second = reverse(slow);
        ListNode* first = head;

        while (second) {
            if (first->val != second->val) return false;
            first = first->next;
            second = second->next;
        }

        return true;
    }

private:
    ListNode* reverse(ListNode* head) {
        ListNode* prev = nullptr;
        while (head) {
            ListNode* nxt = head->next;
            head->next = prev;
            prev = head;
            head = nxt;
        }
        return prev;
    }
};
```

```cpp
class Solution {

            fast = fast->next->next;
            if (slow == fast) return true;

        return false;
    }
};
```

### 题意简述

返回链表入环的第一个节点。

### 这题考什么

考察快慢指针找环入口。

### 生活化理解

两个人在环里相遇后，一个回到起点一起走，再次相遇就是入口。

### 核心思路

先用快慢指针判断是否相遇。相遇后，让一个指针回到头节点，另一个留在相遇点。二者每次走一步，再次相遇的位置就是环入口。

### C++ 代码

```cpp
class Solution {
public:
    ListNode *detectCycle(ListNode *head) {
        ListNode* slow = head;
        ListNode* fast = head;

        while (fast && fast->next) {
            slow = slow->next;
            fast = fast->next->next;
            if (slow == fast) {
                ListNode* p = head;
                while (p != slow) {
                    p = p->next;
                    slow = slow->next;
                }
                return p;
            }
        }

        return nullptr;
    }
};
```

### 题意简述

### 这题考什么

### 生活化理解

像把两队按身高排好的人合成一队，每次选较矮的接到新队伍后面。

### 核心思路

用虚拟头节点 `dummy`，每次接上较小节点。

### C++ 代码

```cpp
class Solution {
public:
    ListNode* mergeTwoLists(ListNode* list1, ListNode* list2) {
        ListNode dummy;
        ListNode* cur = &dummy;

        while (list1 && list2) {
            if (list1->val < list2->val) {
                cur->next = list1;
                list1 = list1->next;
            } else {
                cur->next = list2;
                list2 = list2->next;
            }
            cur = cur->next;
        }

        cur->next = list1 ? list1 : list2;
        return dummy.next;
    }
};
```

## 7. 两数相加

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

两个链表表示逆序数字，返回相加结果链表。

### 这题考什么

考察链表模拟竖式加法和进位。

### 生活化理解

像小学竖式加法，从个位开始加，满十进一。

### 核心思路

像竖式加法，从低位到高位逐位相加，维护进位 `carry`。

### C++ 代码

```cpp
class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        ListNode dummy;
        ListNode* cur = &dummy;
        int carry = 0;

        while (l1 || l2 || carry) {
            int sum = carry;
            if (l1) {
                sum += l1->val;
                l1 = l1->next;
            }
            if (l2) {
                sum += l2->val;
                l2 = l2->next;
            }

            carry = sum / 10;
            cur->next = new ListNode(sum % 10);
            cur = cur->next;
        }

        return dummy.next;
    }
};
```

```cpp
            slow = slow->next;

        slow->next = slow->next->next;
        return dummy.next;

};
```

```cpp
            cur->next = b;
```

## 10. K 个一组翻转链表

链接：https://leetcode.cn/problems/reverse-nodes-in-k-group/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

解决「K 个一组翻转链表」这道 Hot100 经典题。

### 这题考什么

考察对应章节的核心算法能力。

### 生活化理解

把抽象问题转成可维护的状态或结构。

### 核心思路

每次检查后面是否有 k 个节点。有就翻转这 k 个节点，并接回原链表；不足 k 个则保持不变。

### C++ 代码

```cpp
class Solution {
public:
    ListNode* reverseKGroup(ListNode* head, int k) {
        ListNode dummy(0, head);
        ListNode* pre = &dummy;

        while (true) {
            ListNode* end = pre;
            for (int i = 0; i < k && end; i++) {
                end = end->next;
            }
            if (!end) break;

            ListNode* start = pre->next;
            ListNode* nextGroup = end->next;
            end->next = nullptr;

            pre->next = reverse(start);
            start->next = nextGroup;
            pre = start;
        }

        return dummy.next;
    }

private:
    ListNode* reverse(ListNode* head) {
        ListNode* prev = nullptr;

        }

    }
};
```

## 12. 排序链表

链接：https://leetcode.cn/problems/sort-list/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

对链表进行排序。

### 这题考什么

考察链表归并排序。

### 生活化理解

像把长队不断对半拆开，分别排好后再合并。

### 核心思路

链表排序常用归并排序。用快慢指针找到中点，断开链表，分别排序后合并。

### C++ 代码

```cpp
class Solution {
public:
    ListNode* sortList(ListNode* head) {
        if (!head || !head->next) return head;

        ListNode* slow = head;
        ListNode* fast = head->next;
        while (fast && fast->next) {
            slow = slow->next;
            fast = fast->next->next;
        }

        ListNode* mid = slow->next;
        slow->next = nullptr;

        return merge(sortList(head), sortList(mid));
    }

private:
    ListNode* merge(ListNode* a, ListNode* b) {
        ListNode dummy;
        ListNode* cur = &dummy;
        while (a && b) {
            if (a->val < b->val) {
                cur->next = a;
                a = a->next;
            } else {
                cur->next = b;
                b = b->next;
            }
            cur = cur->next;
        }
        cur->next = a ? a : b;
        return dummy.next;
    }
};
```

- 时间复杂度：通常为 O(n)，需要访问每个节点或格子。

### 这题考什么

### 生活化理解

### 核心思路

用小根堆维护每条链表当前最小节点。每弹出一个节点，就把它的下一个节点放入堆。

### C++ 代码

```cpp
class Solution {
public:
    struct Cmp {
        bool operator()(ListNode* a, ListNode* b) {
            return a->val > b->val;
        }
    };

    ListNode* mergeKLists(vector<ListNode*>& lists) {
        priority_queue<ListNode*, vector<ListNode*>, Cmp> pq;
        for (auto node : lists) {
            if (node) pq.push(node);
        }

        ListNode dummy;
        ListNode* cur = &dummy;

        while (!pq.empty()) {
            ListNode* node = pq.top();
            pq.pop();
            cur->next = node;
            cur = cur->next;
            if (node->next) pq.push(node->next);
        }

        return dummy.next;
    }
};
```

## 14. LRU 缓存

链接：https://leetcode.cn/problems/lru-cache/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

解决「LRU 缓存」这道 Hot100 经典题。

### 这题考什么

考察对应章节的核心算法能力。

### 生活化理解

把抽象问题转成可维护的状态或结构。

### 核心思路

LRU 需要同时满足：

- O(1) 查找 key：用 `unordered_map`
- O(1) 移动最近使用节点：用双向链表 `list`

`list<pair<int, int>> cache` 存 `{key, value}`，链表头表示最近使用，尾部表示最久未使用。

### C++ 代码

```cpp
class LRUCache {
private:
    int cap;
    list<pair<int, int>> cache;
    unordered_map<int, list<pair<int, int>>::iterator> mp;

public:
    LRUCache(int capacity) {
        cap = capacity;
    }
    
    int get(int key) {
        if (!mp.count(key)) return -1;
        auto it = mp[key];
        int value = it->second;
        cache.erase(it);
        cache.push_front({key, value});
        mp[key] = cache.begin();
        return value;
    }
    
    void put(int key, int value) {
        if (mp.count(key)) {
            cache.erase(mp[key]);
        } else if (cache.size() == cap) {
            auto last = cache.back();
            mp.erase(last.first);
            cache.pop_back();
        }

        cache.push_front({key, value});
        mp[key] = cache.begin();
    }
};
```
