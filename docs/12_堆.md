# LeetCode Hot 100 - 12 堆 C++ 题解

## 一、堆知识点回顾

堆适合动态维护最大值或最小值。

C++ `priority_queue` 默认是大根堆：

```cpp
priority_queue<int> maxHeap;
```

小根堆：

```cpp
priority_queue<int, vector<int>, greater<int>> minHeap;
```

看到 Top K、数据流中位数、多个有序链表合并，要优先想到堆。

## 1. 数组中的第 K 个最大元素

链接：https://leetcode.cn/problems/kth-largest-element-in-an-array/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

使用堆解决「数组中的第 K 个最大元素」。

### 这题考什么

考察动态维护最大值、最小值或 Top K。

### 生活化理解

像候选池里随时能拿出当前最优候选。

### 解题思路

维护一个大小为 k 的小根堆。堆里始终保留当前最大的 k 个数，堆顶就是第 k 大。

### C++ 代码

```cpp
class Solution {
public:
    int findKthLargest(vector<int>& nums, int k) {
        priority_queue<int, vector<int>, greater<int>> pq;

        for (int x : nums) {
            pq.push(x);
            if (pq.size() > k) pq.pop();
        }

        return pq.top();
    }
};
```

## 2. 前 K 个高频元素

链接：https://leetcode.cn/problems/top-k-frequent-elements/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

使用堆解决「前 K 个高频元素」。

### 这题考什么

考察动态维护最大值、最小值或 Top K。

### 生活化理解

像候选池里随时能拿出当前最优候选。

### 解题思路

先用哈希表统计频率，再用大小为 k 的小根堆维护频率最高的 k 个元素。

### C++ 代码

```cpp
class Solution {
public:
    vector<int> topKFrequent(vector<int>& nums, int k) {
        unordered_map<int, int> cnt;
        for (int x : nums) cnt[x]++;

        auto cmp = [](const pair<int, int>& a, const pair<int, int>& b) {
            return a.second > b.second;
        };

        priority_queue<pair<int, int>, vector<pair<int, int>>, decltype(cmp)> pq(cmp);

        for (auto& item : cnt) {
            pq.push(item);
            if (pq.size() > k) pq.pop();
        }

        vector<int> ans;
        while (!pq.empty()) {
            ans.push_back(pq.top().first);
            pq.pop();
        }

        return ans;
    }
};
```

## 3. 数据流的中位数

链接：https://leetcode.cn/problems/find-median-from-data-stream/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

使用堆解决「数据流的中位数」。

### 这题考什么

考察动态维护最大值、最小值或 Top K。

### 生活化理解

像候选池里随时能拿出当前最优候选。

### 解题思路

用两个堆：

- 大根堆 `small` 存较小的一半
- 小根堆 `large` 存较大的一半

保持 `small.size() >= large.size()` 且最多多一个元素。

### C++ 代码

```cpp
class MedianFinder {
private:
    priority_queue<int> small;
    priority_queue<int, vector<int>, greater<int>> large;

public:
    MedianFinder() {}
    
    void addNum(int num) {
        if (small.empty() || num <= small.top()) {
            small.push(num);
        } else {
            large.push(num);
        }

        if (small.size() > large.size() + 1) {
            large.push(small.top());
            small.pop();
        } else if (large.size() > small.size()) {
            small.push(large.top());
            large.pop();
        }
    }
    
    double findMedian() {
        if (small.size() == large.size()) {
            return (small.top() + large.top()) / 2.0;
        }
        return small.top();
    }
};
```
