# LeetCode Hot 100 - 09 回溯 C++ 题解

## 一、回溯知识点回顾

回溯就是“试一条路，不行就退回来换一条”。它不是暴力乱试，而是有结构地枚举所有可能。

模板：

```cpp
void backtrack(路径, 选择列表) {
    if (结束条件) {
        收集答案;
        return;
    }
    for (选择 : 选择列表) {
        做选择;
        backtrack(...);
        撤销选择;
    }
}
```

三要素：

- 路径：已经选择了什么。
- 选择列表：当前还能选什么。
- 结束条件：什么时候收集答案。


## 二、基础容器/语法解释

回溯题常用全局答案和路径数组。

### path 路径

```cpp
vector<int> path;
path.push_back(x);
path.pop_back();
```

- `push_back`：做选择。
- `pop_back`：撤销选择。

### ans 答案

```cpp
vector<vector<int>> ans;
ans.push_back(path);
```

把当前路径保存到答案中。

### used 数组

```cpp
vector<bool> used(nums.size(), false);
used[i] = true;
used[i] = false;
```

用于标记某个元素是否已经被使用。

### substr

```cpp
s.substr(start, len);
```

从 `start` 开始截取长度为 `len` 的子串，分割回文串常用。

## 1. 全排列

链接：https://leetcode.cn/problems/permutations/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

枚举「全排列」的所有合法方案。

### 这题考什么

考察路径、选择列表、结束条件和撤销选择。

### 生活化理解

像走迷宫，每次选一条路，走不通就退回来换下一条。

### 核心思路

每个位置选择一个没用过的数，直到路径长度等于数组长度。

### C++ 代码

```cpp
class Solution {
public:
    vector<vector<int>> ans;
    vector<int> path;
    vector<bool> used;

    vector<vector<int>> permute(vector<int>& nums) {
        used.assign(nums.size(), false);
        backtrack(nums);
        return ans;
    }

    void backtrack(vector<int>& nums) {
        if (path.size() == nums.size()) {
            ans.push_back(path);
            return;
        }

        for (int i = 0; i < nums.size(); i++) {
            if (used[i]) continue;
            used[i] = true;
            path.push_back(nums[i]);
            backtrack(nums);
            path.pop_back();
            used[i] = false;
        }
    }
};
```

- 时间复杂度：与搜索树规模有关，通常是指数级。
- 空间复杂度：递归深度和路径存储。

class Solution {
public:

        return ans;

    void backtrack(vector<int>& nums, int start) {
        ans.push_back(path);

        for (int i = start; i < nums.size(); i++) {
            path.push_back(nums[i]);
            backtrack(nums, i + 1);
            path.pop_back();
        }
    }
};

## 3. 电话号码的字母组合

链接：https://leetcode.cn/problems/letter-combinations-of-a-phone-number/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

枚举「电话号码的字母组合」的所有合法方案。

### 这题考什么

### 生活化理解

像走迷宫，每次选一条路，走不通就退回来换下一条。

### 核心思路

每个数字对应一组字母，每一层选择当前数字的一个字母。

### C++ 代码

```cpp
class Solution {
public:
    vector<string> ans;
    string path;
    vector<string> mp = {"", "", "abc", "def", "ghi", "jkl", "mno", "pqrs", "tuv", "wxyz"};

    vector<string> letterCombinations(string digits) {
        if (digits.empty()) return ans;
        backtrack(digits, 0);
        return ans;
    }

    void backtrack(string& digits, int idx) {
        if (idx == digits.size()) {
            ans.push_back(path);
            return;
        }

        string letters = mp[digits[idx] - '0'];
        for (char c : letters) {
            path.push_back(c);
            backtrack(digits, idx + 1);
            path.pop_back();
        }
    }
};
```

## 4. 组合总和

链接：https://leetcode.cn/problems/combination-sum/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

枚举「组合总和」的所有合法方案。

### 这题考什么

考察路径、选择列表、结束条件和撤销选择。

### 生活化理解

像走迷宫，每次选一条路，走不通就退回来换下一条。

### 核心思路

每个数字可以重复使用，所以递归时仍然传 `i`，不是 `i + 1`。先排序，当前和超过目标就剪枝。

### C++ 代码

```cpp
class Solution {
public:
    vector<vector<int>> ans;
    vector<int> path;

    vector<vector<int>> combinationSum(vector<int>& candidates, int target) {
        sort(candidates.begin(), candidates.end());
        backtrack(candidates, target, 0, 0);
        return ans;
    }

    void backtrack(vector<int>& candidates, int target, int start, int sum) {
        if (sum == target) {
            ans.push_back(path);
            return;
        }

        for (int i = start; i < candidates.size(); i++) {
            if (sum + candidates[i] > target) break;
            path.push_back(candidates[i]);
            backtrack(candidates, target, i, sum + candidates[i]);
            path.pop_back();
        }
    }
};
```

## 5. 括号生成

链接：https://leetcode.cn/problems/generate-parentheses/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

### 这题考什么

考察路径、选择列表、结束条件和撤销选择。

### 生活化理解

像走迷宫，每次选一条路，走不通就退回来换下一条。

### 核心思路

维护已经使用的左括号和右括号数量：

- 左括号数量小于 n 时，可以放左括号。
- 右括号数量小于左括号数量时，可以放右括号。

### C++ 代码

```cpp
class Solution {
public:
    vector<string> ans;
    string path;

    vector<string> generateParenthesis(int n) {
        backtrack(n, 0, 0);
        return ans;
    }

    void backtrack(int n, int left, int right) {
        if (path.size() == 2 * n) {
            ans.push_back(path);
            return;
        }

        if (left < n) {
            path.push_back('(');
            backtrack(n, left + 1, right);
            path.pop_back();
        }

        if (right < left) {
            path.push_back(')');
            backtrack(n, left, right + 1);
            path.pop_back();
        }
    }
};
```

## 6. 单词搜索

链接：https://leetcode.cn/problems/word-search/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

枚举「单词搜索」的所有合法方案。

### 这题考什么

考察路径、选择列表、结束条件和撤销选择。

### 生活化理解

像走迷宫，每次选一条路，走不通就退回来换下一条。

### 核心思路

从每个格子出发 DFS，匹配当前字符后向四个方向搜索。访问过的格子要临时标记，回溯时恢复。

### C++ 代码

```cpp
class Solution {
public:
    bool exist(vector<vector<char>>& board, string word) {
        int m = board.size();
        int n = board[0].size();

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (dfs(board, word, i, j, 0)) return true;
            }
        }

        return false;
    }

    bool dfs(vector<vector<char>>& board, string& word, int i, int j, int k) {
        if (k == word.size()) return true;
        int m = board.size();
        int n = board[0].size();
        if (i < 0 || i >= m || j < 0 || j >= n || board[i][j] != word[k]) return false;

        char old = board[i][j];
        board[i][j] = '#';

        bool ok = dfs(board, word, i + 1, j, k + 1) ||
                  dfs(board, word, i - 1, j, k + 1) ||
                  dfs(board, word, i, j + 1, k + 1) ||
                  dfs(board, word, i, j - 1, k + 1);

        board[i][j] = old;
        return ok;
    }
};
```

## 7. 分割回文串

链接：https://leetcode.cn/problems/palindrome-partitioning/

标签：本章核心专题

### 这题考什么

### 生活化理解

像走迷宫，每次选一条路，走不通就退回来换下一条。

### 核心思路

枚举从 `start` 开始的所有切割位置，如果当前子串是回文，就加入路径继续切后面。

### C++ 代码

```cpp
class Solution {
public:
    vector<vector<string>> ans;
    vector<string> path;

    vector<vector<string>> partition(string s) {
        backtrack(s, 0);
        return ans;
    }

    void backtrack(string& s, int start) {
        if (start == s.size()) {
            ans.push_back(path);
            return;
        }

        for (int i = start; i < s.size(); i++) {
            if (!isPalindrome(s, start, i)) continue;
            path.push_back(s.substr(start, i - start + 1));
            backtrack(s, i + 1);
            path.pop_back();
        }
    }

    bool isPalindrome(string& s, int l, int r) {
        while (l < r) {
            if (s[l++] != s[r--]) return false;
        }
        return true;
    }
};
```

## 8. N 皇后

链接：https://leetcode.cn/problems/n-queens/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

枚举「N 皇后」的所有合法方案。

### 这题考什么

考察路径、选择列表、结束条件和撤销选择。

### 生活化理解

像走迷宫，每次选一条路，走不通就退回来换下一条。

### 核心思路

每一行放一个皇后，检查列、主对角线、副对角线是否冲突。

对角线判断：

- 主对角线：`row - col`
- 副对角线：`row + col`

### C++ 代码

```cpp
class Solution {
public:
    vector<vector<string>> ans;
    vector<string> board;
    unordered_set<int> cols, diag1, diag2;

    vector<vector<string>> solveNQueens(int n) {
        board.assign(n, string(n, '.'));
        backtrack(0, n);
        return ans;
    }

    void backtrack(int row, int n) {
        if (row == n) {
            ans.push_back(board);
            return;
        }

        for (int col = 0; col < n; col++) {
            if (cols.count(col) || diag1.count(row - col) || diag2.count(row + col)) continue;

            board[row][col] = 'Q';
            cols.insert(col);
            diag1.insert(row - col);
            diag2.insert(row + col);

            backtrack(row + 1, n);

            board[row][col] = '.';
            cols.erase(col);
            diag1.erase(row - col);
            diag2.erase(row + col);
        }
    }
};
```
