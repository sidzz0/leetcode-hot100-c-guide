# LeetCode Hot 100 - 16 技巧题 C++ 题解

## 一、技巧题知识点回顾

技巧题不是新算法，而是对已有知识的特殊使用。

常见技巧：

- 位运算异或抵消。
- Boyer-Moore 投票法。
- 三指针分区。
- 字典序下一个排列。
- 快慢指针把数组看成链表。


## 二、基础容器/语法解释

技巧题常用一些特殊语法和变量维护。

### 异或

```cpp
ans ^= x;
```

异或性质：

- `x ^ x = 0`
- `x ^ 0 = x`

### 三指针

```cpp
int left = 0, i = 0, right = nums.size() - 1;
```

用于把数组分成多个区域。

### reverse

```cpp
reverse(nums.begin() + i + 1, nums.end());
```

反转后缀，常用于下一个排列。

### 快慢指针数组写法

```cpp
slow = nums[slow];
fast = nums[nums[fast]];
```

把数组值当作下一个下标，从而把数组看成链表。

### 生活化理解

### 核心思路

异或性质：

```text
x ^ x = 0
x ^ 0 = x
```

所有数字异或一遍，成对数字抵消，剩下的就是只出现一次的数字。

### C++ 代码

```cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int ans = 0;
        for (int x : nums) ans ^= x;
        return ans;
    }
};
```

### 生活化理解

把抽象问题转成可维护的状态或结构。

### 核心思路

Boyer-Moore 投票法。把多数元素和其他元素两两抵消，最后剩下的一定是多数元素。

### C++ 代码

```cpp
class Solution {
public:
    int majorityElement(vector<int>& nums) {
        int candidate = 0;
        int count = 0;

        for (int x : nums) {
            if (count == 0) candidate = x;
            count += (x == candidate) ? 1 : -1;
        }

        return candidate;
    }
};
```

## 3. 颜色分类

链接：https://leetcode.cn/problems/sort-colors/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

解决「颜色分类」这道 Hot100 经典题。

### 这题考什么

### 生活化理解

把抽象问题转成可维护的状态或结构。

### 核心思路

荷兰国旗问题。维护三个区域：

- `[0, left)` 是 0
- `[left, i)` 是 1
- `(right, n-1]` 是 2

### C++ 代码

```cpp
class Solution {
public:
    void sortColors(vector<int>& nums) {
        int left = 0;
        int i = 0;
        int right = nums.size() - 1;

        while (i <= right) {
            if (nums[i] == 0) {
                swap(nums[i], nums[left]);
                left++;
                i++;
            } else if (nums[i] == 2) {
                swap(nums[i], nums[right]);
                right--;
            } else {
                i++;
            }
        }
    }
};
```

## 4. 下一个排列

链接：https://leetcode.cn/problems/next-permutation/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

解决「下一个排列」这道 Hot100 经典题。

### 这题考什么

考察对应章节的核心算法能力。

### 生活化理解

把抽象问题转成可维护的状态或结构。

### 核心思路

从右往左找第一个下降位置 `i`，即 `nums[i] < nums[i+1]`。再从右往左找第一个大于 `nums[i]` 的数交换，最后反转 `i+1` 后面的部分。

### C++ 代码

```cpp
class Solution {
public:
    void nextPermutation(vector<int>& nums) {
        int n = nums.size();
        int i = n - 2;

        while (i >= 0 && nums[i] >= nums[i + 1]) i--;

        if (i >= 0) {
            int j = n - 1;
            while (nums[j] <= nums[i]) j--;
            swap(nums[i], nums[j]);
        }

        reverse(nums.begin() + i + 1, nums.end());
    }
};
```

## 5. 寻找重复数

链接：https://leetcode.cn/problems/find-the-duplicate-number/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

解决「寻找重复数」这道 Hot100 经典题。

### 这题考什么

考察对应章节的核心算法能力。

### 生活化理解

把抽象问题转成可维护的状态或结构。

### 核心思路

把数组看成链表：下标指向 `nums[index]`。因为有重复数，所以一定有环，重复数就是环入口。

用快慢指针找环入口。

### C++ 代码

```cpp
class Solution {
public:
    int findDuplicate(vector<int>& nums) {
        int slow = nums[0];
        int fast = nums[0];

        do {
            slow = nums[slow];
            fast = nums[nums[fast]];
        } while (slow != fast);

        int p = nums[0];
        while (p != slow) {
            p = nums[p];
            slow = nums[slow];
        }

        return p;
    }
};
```
