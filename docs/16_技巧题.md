## 1. 只出现一次的数字

### 思路

异或性质：

```text
x ^ x = 0
x ^ 0 = x
```

所有数字异或一遍，成对数字抵消，剩下的就是只出现一次的数字。

### C++ 代码

```cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int ans = 0;
        for (int x : nums) ans ^= x;
        return ans;
    }
};
```

## 2. 多数元素

### 思路

Boyer-Moore 投票法。把多数元素和其他元素两两抵消，最后剩下的一定是多数元素。

### C++ 代码

```cpp
class Solution {
public:
    int majorityElement(vector<int>& nums) {
        int candidate = 0;
        int count = 0;

        for (int x : nums) {
            if (count == 0) candidate = x;
            count += (x == candidate) ? 1 : -1;
        }

        return candidate;
    }
};
```

## 3. 颜色分类

### 思路

荷兰国旗问题。维护三个区域：

- `[0, left)` 是 0
- `[left, i)` 是 1
- `(right, n-1]` 是 2

### C++ 代码

```cpp
class Solution {
public:
    void sortColors(vector<int>& nums) {
        int left = 0;
        int i = 0;
        int right = nums.size() - 1;

        while (i <= right) {
            if (nums[i] == 0) {
                swap(nums[i], nums[left]);
                left++;
                i++;
            } else if (nums[i] == 2) {
                swap(nums[i], nums[right]);
                right--;
            } else {
                i++;
            }
        }
    }
};
```

易错点：遇到 2 和右边交换后，`i` 不能立刻加一，因为换过来的数还没检查。

## 4. 下一个排列

### 思路

从右往左找第一个下降位置 `i`，即 `nums[i] < nums[i+1]`。再从右往左找第一个大于 `nums[i]` 的数交换，最后反转 `i+1` 后面的部分。

### C++ 代码

```cpp
class Solution {
public:
    void nextPermutation(vector<int>& nums) {
        int n = nums.size();
        int i = n - 2;

        while (i >= 0 && nums[i] >= nums[i + 1]) i--;

        if (i >= 0) {
            int j = n - 1;
            while (nums[j] <= nums[i]) j--;
            swap(nums[i], nums[j]);
        }

        reverse(nums.begin() + i + 1, nums.end());
    }
};
```

## 5. 寻找重复数

### 思路

把数组看成链表：下标指向 `nums[index]`。因为有重复数，所以一定有环，重复数就是环入口。

用快慢指针找环入口。

### C++ 代码

```cpp
class Solution {
public:
    int findDuplicate(vector<int>& nums) {
        int slow = nums[0];
        int fast = nums[0];

        do {
            slow = nums[slow];
            fast = nums[nums[fast]];
        } while (slow != fast);

        int p = nums[0];
        while (p != slow) {
            p = nums[p];
            slow = nums[slow];
        }

        return p;
    }
};
```
