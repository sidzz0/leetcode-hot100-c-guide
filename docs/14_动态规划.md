# LeetCode Hot 100 - 14 动态规划 C++ 题解

## 一、动态规划知识点回顾

动态规划适合处理有重叠子问题、状态可以从更小状态转移而来的题。

DP 五步法：

1. 定义 `dp` 含义。
2. 写状态转移方程。
3. 初始化。
4. 确定遍历顺序。
5. 举例验证。

常见类型：

- 线性 DP
- 背包 DP
- 字符串切割 DP
- 子序列 DP


## 二、基础容器/语法解释

动态规划常用一维或二维数组。

### 一维 dp

```cpp
vector<int> dp(n + 1, 0);
```

适合线性状态或背包问题。

### 二维 dp

```cpp
vector<vector<int>> dp(m + 1, vector<int>(n + 1, 0));
```

适合两个维度共同决定状态的问题。

### vector<bool>

```cpp
vector<bool> dp(target + 1, false);
```

常用于表示“能否凑出某个状态”。

### accumulate

```cpp
int sum = accumulate(nums.begin(), nums.end(), 0);
```

计算数组总和，背包类题常用。

使用前需要包含：

```cpp
#include <numeric>
```

```cpp
class Solution {
public:
    int rob(vector<int>& nums) {
        int prev2 = 0, prev1 = 0;

        for (int x : nums) {
            int cur = max(prev1, prev2 + x);
            prev2 = prev1;
            prev1 = cur;
        }

        return prev1;
    }
};
```

### C++ 代码

```cpp
class Solution {
public:
    int numSquares(int n) {
        vector<int> dp(n + 1, INT_MAX);
        dp[0] = 0;

        for (int i = 1; i <= n; i++) {
            for (int j = 1; j * j <= i; j++) {
                dp[i] = min(dp[i], dp[i - j * j] + 1);
            }
        }

        return dp[n];
    }
};
```

```cpp
class Solution {

    int coinChange(vector<int>& coins, int amount) {
        vector<int> dp(amount + 1, amount + 1);
        dp[0] = 0;

        for (int i = 1; i <= amount; i++) {
            for (int coin : coins) {
                if (i >= coin) {
                    dp[i] = min(dp[i], dp[i - coin] + 1);
                }
            }
        }

    }
};
```

## 6. 单词拆分

链接：https://leetcode.cn/problems/word-break/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

用动态规划解决「单词拆分」。

### 这题考什么

考察状态定义、状态转移和初始化。

### 生活化理解

像填表格，后面的答案依赖前面已经算好的答案。

### 核心思路

`dp[i]` 表示 `s[0..i)` 是否可以被拆分。枚举切割点 `j`，如果 `dp[j]` 为真且 `s[j..i)` 在字典中，则 `dp[i]` 为真。

### C++ 代码

```cpp
class Solution {
public:
    bool wordBreak(string s, vector<string>& wordDict) {
        unordered_set<string> words(wordDict.begin(), wordDict.end());
        vector<bool> dp(s.size() + 1, false);
        dp[0] = true;

        for (int i = 1; i <= s.size(); i++) {
            for (int j = 0; j < i; j++) {
                if (dp[j] && words.count(s.substr(j, i - j))) {
                    dp[i] = true;
                    break;
                }
            }
        }

        return dp[s.size()];
    }
};
```

用动态规划解决「乘积最大子数组」。

### 这题考什么

### 生活化理解

像填表格，后面的答案依赖前面已经算好的答案。

### 核心思路

负数会让最大变最小、最小变最大，所以同时维护当前最大乘积和最小乘积。

### C++ 代码

```cpp
class Solution {
public:
    int maxProduct(vector<int>& nums) {
        int maxVal = nums[0];
        int minVal = nums[0];
        int ans = nums[0];

        for (int i = 1; i < nums.size(); i++) {
            int x = nums[i];
            if (x < 0) swap(maxVal, minVal);
            maxVal = max(x, maxVal * x);
            minVal = min(x, minVal * x);
            ans = max(ans, maxVal);
        }

        return ans;
    }
};
```

### 核心思路

如果总和是奇数，不可能分割。目标变成能否从数组中选一些数凑出 `sum / 2`，这是 01 背包。

### C++ 代码

```cpp
class Solution {
public:
    bool canPartition(vector<int>& nums) {
        int sum = accumulate(nums.begin(), nums.end(), 0);
        if (sum % 2) return false;

        int target = sum / 2;
        vector<bool> dp(target + 1, false);
        dp[0] = true;

        for (int x : nums) {
            for (int j = target; j >= x; j--) {
                dp[j] = dp[j] || dp[j - x];
            }
        }

        return dp[target];
    }
};
```

## 10. 最长有效括号

链接：https://leetcode.cn/problems/longest-valid-parentheses/

标签：本章核心专题  
难度：Hot100 高频  
重要程度：秋招重点

### 题意简述

用动态规划解决「最长有效括号」。

### 这题考什么

考察状态定义、状态转移和初始化。

### 生活化理解

像填表格，后面的答案依赖前面已经算好的答案。

### 核心思路

`dp[i]` 表示以 `s[i]` 结尾的最长有效括号长度。只有 `s[i] == ')'` 时才可能形成有效括号。

如果前一个是 `(`：

```text
...()
```

如果前一个是 `)`，需要看前面是否有能匹配当前右括号的左括号。

### C++ 代码

```cpp
class Solution {
public:
    int longestValidParentheses(string s) {
        int n = s.size();
        vector<int> dp(n, 0);
        int ans = 0;

        for (int i = 1; i < n; i++) {
            if (s[i] == ')') {
                if (s[i - 1] == '(') {
                    dp[i] = (i >= 2 ? dp[i - 2] : 0) + 2;
                } else {
                    int j = i - dp[i - 1] - 1;
                    if (j >= 0 && s[j] == '(') {
                        dp[i] = dp[i - 1] + 2 + (j >= 1 ? dp[j - 1] : 0);
                    }
                }
                ans = max(ans, dp[i]);
            }
        }

        return ans;
    }
};
```
