## 一、普通数组知识点回顾

数组题看起来朴素，但常考这些能力：

- 遍历时维护状态
- 排序后合并
- 原地修改
- 前缀/后缀信息
- 利用下标表达数字关系

常用 C++ 操作：

```cpp
sort(nums.begin(), nums.end());
reverse(nums.begin(), nums.end());
vector<int> ans(n, 1);
```

`reverse(l, r)` 会反转区间 `[l, r)`，也就是左闭右开。

## 1. 最大子数组和

链接：https://leetcode.cn/problems/maximum-subarray/

### 题意简述

找出数组中和最大的连续子数组，并返回最大和。

### 生活化理解

像连续几天收益，有赚有亏。你想找一段连续时间，让总收益最大。如果前面累计收益已经变成负数，它只会拖累后面，不如从今天重新开始。

### 思路：Kadane 算法

维护：

- `cur`：以当前元素结尾的最大子数组和
- `ans`：全局最大子数组和

状态转移：

```cpp
cur = max(nums[i], cur + nums[i]);
```

意思是：当前元素要么单独开头，要么接在前面的连续子数组后面。

### C++ 代码

```cpp
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int cur = nums[0];
        int ans = nums[0];

        for (int i = 1; i < nums.size(); i++) {
            cur = max(nums[i], cur + nums[i]);
            ans = max(ans, cur);
        }

        return ans;
    }
};
```

### 复杂度与易错点

- 时间复杂度：O(n)
- 空间复杂度：O(1)
- 易错点：全是负数时答案是最大的那个负数，所以初始化为 `nums[0]`。

## 2. 合并区间

链接：https://leetcode.cn/problems/merge-intervals/

### 题意简述

给多个区间，合并所有重叠区间。

### 生活化理解

像整理会议室预约时间。先按开始时间排序，后面的会议如果开始时间不晚于当前会议结束时间，就可以合并成一个更长的时间段。

### 思路

先按左端点排序。维护结果数组 `ans`：

- 如果当前区间和最后一个结果区间不重叠，直接加入。
- 如果重叠，更新最后一个区间的右端点。

### C++ 代码

```cpp
class Solution {
public:
    vector<vector<int>> merge(vector<vector<int>>& intervals) {
        sort(intervals.begin(), intervals.end());
        vector<vector<int>> ans;

        for (auto& cur : intervals) {
            if (ans.empty() || ans.back()[1] < cur[0]) {
                ans.push_back(cur);
            } else {
                ans.back()[1] = max(ans.back()[1], cur[1]);
            }
        }

        return ans;
    }
};
```

### 复杂度与易错点

- 时间复杂度：O(n log n)
- 空间复杂度：O(n)
- 易错点：重叠条件是 `ans.back()[1] >= cur[0]`。

## 3. 轮转数组

链接：https://leetcode.cn/problems/rotate-array/

### 题意简述

把数组向右轮转 `k` 步。

### 生活化理解

像一排人整体向右挪，末尾的人从前面补回来。右移 `k` 次，其实只和 `k % n` 有关。

### 思路：三次反转

例如：

```text
1 2 3 4 5 6 7, k = 3
```

整体反转：

```text
7 6 5 4 3 2 1
```

反转前 k 个：

```text
5 6 7 4 3 2 1
```

反转剩余部分：

```text
5 6 7 1 2 3 4
```

### C++ 代码

```cpp
class Solution {
public:
    void rotate(vector<int>& nums, int k) {
        int n = nums.size();
        k %= n;

        reverse(nums.begin(), nums.end());
        reverse(nums.begin(), nums.begin() + k);
        reverse(nums.begin() + k, nums.end());
    }
};
```

### 复杂度与易错点

- 时间复杂度：O(n)
- 空间复杂度：O(1)
- 易错点：先 `k %= n`。
- 易错点：`reverse` 是左闭右开区间。

## 4. 除自身以外数组的乘积

链接：https://leetcode.cn/problems/product-of-array-except-self/

### 题意简述

返回一个数组，`ans[i]` 等于除了 `nums[i]` 以外所有元素的乘积。

不能使用除法。

### 生活化理解

每个位置需要左边所有数的乘积和右边所有数的乘积。就像你站在队伍里，只关心自己左边一组和右边一组的总贡献。

### 思路：前缀乘积 + 后缀乘积

先从左到右，让 `ans[i]` 存 `i` 左边所有数乘积。  
再从右到左，用变量 `right` 表示右边所有数乘积，乘到 `ans[i]` 上。

### C++ 代码

```cpp
class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        int n = nums.size();
        vector<int> ans(n, 1);

        int left = 1;
        for (int i = 0; i < n; i++) {
            ans[i] = left;
            left *= nums[i];
        }

        int right = 1;
        for (int i = n - 1; i >= 0; i--) {
            ans[i] *= right;
            right *= nums[i];
        }

        return ans;
    }
};
```

### 复杂度与易错点

- 时间复杂度：O(n)
- 空间复杂度：O(1)，不计返回数组
- 易错点：`left` 和 `right` 初始值都是 1。

## 5. 缺失的第一个正数

链接：https://leetcode.cn/problems/first-missing-positive/

### 题意简述

找出数组中缺失的最小正整数，要求 O(n) 时间、O(1) 额外空间。

### 生活化理解

如果数组长度是 n，那么答案只可能在 `1` 到 `n + 1` 之间。我们希望把数字 `x` 放到下标 `x - 1` 的位置。最后谁没站在自己该站的位置，谁就是缺失的第一个正数。

### 思路：原地哈希

遍历数组，把每个在 `[1, n]` 范围内的数放到正确位置：

```text
数字 1 放下标 0
数字 2 放下标 1
数字 x 放下标 x - 1
```

最后再次遍历，如果 `nums[i] != i + 1`，答案就是 `i + 1`。

### C++ 代码

```cpp
class Solution {
public:
    int firstMissingPositive(vector<int>& nums) {
        int n = nums.size();

        for (int i = 0; i < n; i++) {
            while (nums[i] >= 1 && nums[i] <= n &&
                   nums[nums[i] - 1] != nums[i]) {
                swap(nums[i], nums[nums[i] - 1]);
            }
        }

        for (int i = 0; i < n; i++) {
            if (nums[i] != i + 1) {
                return i + 1;
            }
        }

        return n + 1;
    }
};
```

### 复杂度与易错点

- 时间复杂度：O(n)
- 空间复杂度：O(1)
- 易错点：`while` 不是 `if`，换过来的数还要继续检查。
- 易错点：要判断目标位置是否已经是同一个数，避免死循环。
