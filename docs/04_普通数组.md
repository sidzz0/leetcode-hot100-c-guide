# LeetCode Hot 100 - 04 普通数组 C++ 题解

## 一、普通数组知识点回顾

数组题常考的不是“数组是什么”，而是遍历顺序、边界处理、状态维护和原地修改。

常见技巧：

- 当前状态维护：最大子数组和。
- 排序后处理：合并区间。
- 原地操作：轮转数组、缺失的第一个正数。
- 前缀/后缀：除自身以外数组的乘积。

常用 C++ 函数：

```cpp
sort(nums.begin(), nums.end());
reverse(nums.begin(), nums.end());
swap(nums[i], nums[j]);
accumulate(nums.begin(), nums.end(), 0);
```

`reverse(l, r)` 反转的是左闭右开区间 `[l, r)`。


## 二、基础容器/语法解释

本章常用数组操作：

```cpp
sort(nums.begin(), nums.end());
reverse(nums.begin(), nums.end());
swap(nums[i], nums[j]);
vector<int> ans(n, 1);
```

### sort

```cpp
sort(nums.begin(), nums.end());
```

表示将数组从小到大排序。

### reverse

```cpp
reverse(nums.begin(), nums.begin() + k);
```

反转左闭右开区间 `[begin, begin + k)`。

### swap

```cpp
swap(nums[i], nums[j]);
```

交换两个位置的元素，原地哈希和数组调整题经常使用。

### vector 初始化

```cpp
vector<int> ans(n, 1);
```

创建长度为 `n` 的数组，每个元素初始化为 `1`。

## 1. 最大子数组和

链接：https://leetcode.cn/problems/maximum-subarray/

标签：数组、动态规划  
难度：Medium  
重要程度：线性 DP 入门

### 题意简述

找出和最大的连续子数组。

### 这题考什么

维护以当前位置结尾的最大连续和。

### 生活化理解

像连续几天收益，如果前面累计已经亏损，就不要带着亏损继续，从今天重新开始。

### 核心思路

`cur = max(nums[i], cur + nums[i])`，表示当前位置要么自己开头，要么接在前面；`ans` 记录全局最大。

### C++ 代码

```cpp
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int cur = nums[0], ans = nums[0];
        for (int i = 1; i < nums.size(); i++) {
            cur = max(nums[i], cur + nums[i]);
            ans = max(ans, cur);
        }
        return ans;
    }
};
```

### 复杂度分析

- 时间复杂度：O(n)。
- 空间复杂度：O(1)。

### 易错点

- 初始化要用 `nums[0]`，避免全负数出错。
- 题目要求连续子数组，不是子序列。

### 面试表达

维护以当前元素结尾的最大子数组和，如果前面的和为负就重新开始。

### 复刷要求

- 能解释全负数情况。
- 能写出 O(1) 空间版本。

## 2. 合并区间

链接：https://leetcode.cn/problems/merge-intervals/

标签：数组、排序  
难度：Medium  
重要程度：区间题基础

### 题意简述

合并所有重叠区间。

### 这题考什么

排序后维护当前已合并区间。

### 生活化理解

像整理会议室预约，先按开始时间排好，能接上的就合并成一段。

### 核心思路

先按左端点排序。若当前区间左端点大于结果最后区间右端点，说明不重叠；否则更新右端点最大值。

### C++ 代码

```cpp
class Solution {
public:
    vector<vector<int>> merge(vector<vector<int>>& intervals) {
        sort(intervals.begin(), intervals.end());
        vector<vector<int>> ans;
        for (auto& cur : intervals) {
            if (ans.empty() || ans.back()[1] < cur[0]) ans.push_back(cur);
            else ans.back()[1] = max(ans.back()[1], cur[1]);
        }
        return ans;
    }
};
```

### 复杂度分析

- 时间复杂度：O(n log n)。
- 空间复杂度：O(n)。

### 易错点

- 重叠条件是 `ans.back()[1] >= cur[0]`。
- 排序默认按第一个元素，再按第二个元素。

### 面试表达

先排序，再扫描维护结果数组的最后一个区间；重叠则扩展右端点，不重叠则加入新区间。

### 复刷要求

- 能说清楚为什么要先排序。

## 3. 轮转数组

链接：https://leetcode.cn/problems/rotate-array/

标签：数组、双指针  
难度：Medium  
重要程度：原地操作经典题

### 题意简述

将数组向右轮转 k 步。

### 这题考什么

考察原地反转和取模。

### 生活化理解

像一排人向右移动，末尾的人从前面补回来；移动 n 步等于没动。

### 核心思路

先 `k %= n`。三次反转：整体反转、反转前 k 个、反转剩余部分。

### C++ 代码

```cpp
class Solution {
public:
    void rotate(vector<int>& nums, int k) {
        int n = nums.size();
        k %= n;
        reverse(nums.begin(), nums.end());
        reverse(nums.begin(), nums.begin() + k);
        reverse(nums.begin() + k, nums.end());
    }
};
```

### 复杂度分析

- 时间复杂度：O(n)。
- 空间复杂度：O(1)。

### 易错点

- 必须先 `k %= n`。
- `reverse` 是左闭右开区间。

### 面试表达

右轮转可以通过三次反转完成，整体反转后，前 k 个和后 n-k 个分别恢复顺序。

### 复刷要求

- 手推一次 `[1,2,3,4,5,6,7], k=3`。

## 4. 除自身以外数组的乘积

链接：https://leetcode.cn/problems/product-of-array-except-self/

标签：数组、前缀积  
难度：Medium  
重要程度：前后缀经典题

### 题意简述

返回数组，使每个位置等于除自己以外所有数的乘积。

### 这题考什么

考察前缀乘积和后缀乘积，且不能用除法。

### 生活化理解

每个人的贡献来自自己左边所有人和右边所有人，自己不参与。

### 核心思路

先让 `ans[i]` 存左侧乘积，再从右往左维护 `right`，乘上右侧乘积。

### C++ 代码

```cpp
class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        int n = nums.size();
        vector<int> ans(n, 1);
        int left = 1;
        for (int i = 0; i < n; i++) {
            ans[i] = left;
            left *= nums[i];
        }
        int right = 1;
        for (int i = n - 1; i >= 0; i--) {
            ans[i] *= right;
            right *= nums[i];
        }
        return ans;
    }
};
```

### 复杂度分析

- 时间复杂度：O(n)。
- 空间复杂度：O(1)，不计返回数组。

### 易错点

- `left` 和 `right` 初始值都是 1。
- 不能用除法。

### 面试表达

每个位置答案等于左边乘积乘右边乘积，用返回数组先存左乘积，再倒序乘右乘积。

### 复刷要求

- 能解释为什么空间是 O(1)。

## 5. 缺失的第一个正数

链接：https://leetcode.cn/problems/first-missing-positive/

标签：数组、原地哈希  
难度：Hard  
重要程度：数组原地哈希高频题

### 题意简述

找出缺失的最小正整数，要求 O(n) 时间和 O(1) 额外空间。

### 这题考什么

考察把数组下标当哈希桶。

### 生活化理解

如果数字 x 在 1 到 n 之间，就应该站到 x-1 这个位置。最后第一个没站对的位置就是答案。

### 核心思路

答案只可能是 1 到 n+1。遍历数组，把范围内的数字放到对应下标；最后检查 `nums[i] == i+1`。

### C++ 代码

```cpp
class Solution {
public:
    int firstMissingPositive(vector<int>& nums) {
        int n = nums.size();
        for (int i = 0; i < n; i++) {
            while (nums[i] >= 1 && nums[i] <= n &&
                   nums[nums[i] - 1] != nums[i]) {
                swap(nums[i], nums[nums[i] - 1]);
            }
        }
        for (int i = 0; i < n; i++) {
            if (nums[i] != i + 1) return i + 1;
        }
        return n + 1;
    }
};
```

### 复杂度分析

- 时间复杂度：O(n)。
- 空间复杂度：O(1)。

### 易错点

- 用 `while`，不是 `if`。
- 要判断目标位置不是同一个数，避免重复数字死循环。

### 面试表达

把数字 x 放到下标 x-1，完成原地哈希。最后第一个不满足 `nums[i] == i+1` 的位置就是缺失值。

### 复刷要求

- 重点复刷，面试容易考。
